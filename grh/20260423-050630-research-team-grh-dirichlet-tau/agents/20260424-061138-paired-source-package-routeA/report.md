## Claim

The cleanest exact compact-interval paired source-package theorem candidate, stopping before exact slot realization, is the following theorem-facing statement.

**Candidate theorem (compact-interval paired source package, pre-slot form).** Let `chi` be a primitive Dirichlet character and define
`Xi_chi(s)=Lambda(s,chi)Lambda(s,bar chi)` and
`Phi_chi^pair(s)=Xi_chi(2s-1)/Xi_chi(2s)`.
Fix a compact interval `I` in the strip-edge parameter. Then there exists a localized paired source package on `I` consisting of:

1. a phase/quotient normalization built from `Phi_chi^pair`;
2. a one-zero strip-edge contribution for each zero paired with its contragredient mate;
3. a compact-interval summation/regularization formula decomposing the localized paired quantity into the summed one-zero contributions plus a single unified background term `B_chi^pair`;
4. an explicit multiplicity convention for repeated zeros within this compact-interval formula.

The theorem asserts only this localized decomposition and bookkeeping package on `I`.

**Inside the theorem.** The paired quotient shell, the local one-zero kernel shape upstairs, the existence of a compact-interval decomposition formula, a single theorem-facing background term `B_chi^pair`, and explicit multiplicity bookkeeping.

**Outside the theorem.** Any claim that the localized paired source term is already the manuscript's exact local value-channel `S(m)` coefficient, any exact identification of the background-subtracted term with that slot, and any exact realization of whitening/positivity as a consequence of source data alone.

**Proved / conditional / missing.**

- **Proved:** the safest theorem-facing object is the paired quotient `Phi_chi^pair`; pairing removes the residual root-number asymmetry; the zeta source package supplies only a formal blueprint for quotient shell, one-zero kernel shape, and localized compact-interval packaging.
- **Conditional on a paired source theorem:** one may state a local whitening/positivity hypothesis block explicitly, naming microscopic holomorphy of corrected blocks, same-point positivity/nondegeneracy, existence of holomorphic inverse-square-root whitening, and existence of the first paired value-channel derivative.
- **Missing:** exact paired `S(m)`-slot realization; proof that the compact-interval package transfers from the paired source to the exact manuscript local slot; full unified background identification in theorem-ready form; compact-interval convergence/regularization closure; multiplicity closure strong enough for exact slot realization.

Honest verdict: this is the cleanest theorem candidate from current scope alone, but it remains a localized blueprint rather than a closed exact-source theorem.
