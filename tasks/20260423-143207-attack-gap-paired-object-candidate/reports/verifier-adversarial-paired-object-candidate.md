1. `\Xi_\chi(s)=\Lambda(s,\chi)\Lambda(s,\bar\chi)` is the cleanest **raw paired completed precursor**, but not the cleanest theorem-facing paired object by itself.
2. The cleanest theorem-facing paired object from current scope alone is the strip-edge quotient built from that precursor,
   \[
   \Phi^{\mathrm{pair}}_\chi(s):=\frac{\Xi_\chi(2s-1)}{\Xi_\chi(2s)}
   =\phi_\chi(s)\phi_{\bar\chi}(s),
   \]
   because the current program is organized around quotient/source data, not around a bare completed product.
3. Even this quotient-level choice is only a candidate-selection result. From current scope alone, it does **not** prove exact local realization, canonical normalization, or material shrinkage of the remaining theorem burden.

`For a primitive complex Dirichlet character \chi, from current scope alone the conservative exact-source target is a paired strip-edge quotient/source theorem built from the self-paired completed object \Xi_\chi(s)=\Lambda(s,\chi)\Lambda(s,\bar\chi), proving on singularity-free compact intervals an exact decomposition q^{\mathrm{pair}}_\chi=B^{\mathrm{pair}}_\chi+S^{\mathrm{pair}}_\chi in which S^{\mathrm{pair}}_\chi is the background-subtracted scalar occupying the manuscript-style leading local value-channel slot.`

- weaken `pairing fixes the source theorem` to `pairing gives the conservative front-end target from current scope alone`;

- weaken any `canonical realization` wording to `candidate theorem target`.
