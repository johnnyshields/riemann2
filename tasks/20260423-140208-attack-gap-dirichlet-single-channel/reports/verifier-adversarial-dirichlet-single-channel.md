# verifier-adversarial-dirichlet-single-channel

- Claim

  The updated Dirichlet-single-channel conclusion is still only safe as a scoped ranking statement. The paired `((\chi,\bar\chi))` route still deserves to be called the conservative scalar-amplitude target **from current scope alone**, because the single completed quotient has been upgraded only to a conditional phase-channel candidate plus conditional upstairs kernel positivity, not to an exact realization of the manuscript's scalar slot `S(m)` or of the downstream corrected whitening / transverse package. The strongest safe Dirichlet note wording is therefore: `From current scope, the paired ((chi, bar chi)) route remains the more conservative first target for realizing a real nonnegative scalar in the manuscript's exact S-slot; the single completed quotient is a conditional phase-channel candidate and, conditional on a family-specific quotient/source theorem, yields the universal one-zero upstairs kernel and positivity, but not yet the local value-channel realization.` Claims equating quotient positivity with scalar-amplitude realization, or stating the paired route is intrinsically safer without scope, need weakening.

- Status

  open

- Evidence

  Proved:

  - The manuscript does not use an arbitrary positive quotient-visible scalar. It uses the exact background-subtracted value scale `S(m)=q_\zeta(m)-B_\zeta(m)` and inserts that exact scalar into the corrected local deformation `\Delta_\zeta(m,\sigma)=S(m)A_{\val}(m,\sigma)+R_\zeta(m,\sigma)` (`paper/proof_of_rh.tex:273-288`, `1497-1565`). Any Dirichlet wording that slides from `positive quotient object` to `the manuscript scalar` is too strong.
  - The manuscript also separates the scalar-amplitude step from the remaining package: denominator comparability and omitted-smooth holomorphy (`paper/proof_of_rh.tex:856-903`), corrected holomorphic whitening (`1392-1458`), and the odd transverse scalar with boundary estimate and odd microscopic expansion (`2214-2322`). In particular `\Phi_K(A_{\val})=0`, so the transverse cancellation channel is not the same thing as the scalar-amplitude channel (`567-599`).
  - `notes/quotient_phase.md` explicitly says the completed quotient `\phi_F(s)=\Lambda_F(2s-1)/\Lambda_F(2s)` is a serious boundary phase candidate only, and does **not** supply a realized positive `S(m)`-analogue, leading value-channel coefficient theorem, denominator comparability, corrected whitening, microscopic boundary control, or any localization theorem (`grh/20260423-050630-research-team-grh-dirichlet-tau/notes/quotient_phase.md:8-36`). Its own Dirichlet stance keeps the paired route as the more conservative scalar-amplitude target (`38-43`).
  - `notes/universal_source_kernel.md` gives a further scoped upgrade, not a slot-realization theorem: conditional on a family-specific quotient/source theorem, the single-zero quotient-log-derivative kernel and its positivity are universal upstairs, while source identification, background subtraction, convergence, and placement into the exact manuscript `S(m)` slot remain family-specific (`grh/20260423-050630-research-team-grh-dirichlet-tau/notes/universal_source_kernel.md:8-45`). So the single route narrowed the gap, but only upstairs.
  - `notes/positive_s_implication.md` closes only the conditional branch `if the family theorem yields the exact S-slot scalar`. In that exact scope the existing manuscript algebra already supplies the leading-channel coefficient theorem, but quotient/phase data alone does not, and remainder dominance, odd/transverse structure, and family-specific denominator/whitening/boundary control remain open (`grh/20260423-050630-research-team-grh-dirichlet-tau/notes/positive_s_implication.md:8-50`).
  - The current Dirichlet note already reflects this separation: the paired route is `the more conservative first scalar target`, while the rotated single completed channel is `supported only as a conditional phase-channel candidate`; after the quotient-phase attack it remains `the conservative scalar-amplitude target` (`grh/20260423-050630-research-team-grh-dirichlet-tau/notes/dirichlet_channel.md:14-24`, `45-77`). That wording is safe only because it is already scoped to the current evidence base rather than advertised as an intrinsic theorem about all future realizations.
  - Route A correctly isolates the load-bearing ambiguity: if `single-channel quotient source theorem` already includes exact identification of the manuscript slot scalar, then the paired route is no longer needed for positivity of that exact slot; if it means only an upstairs quotient/source decomposition, then the paired route remains the conservative scalar-slot target (`tasks/20260423-140208-attack-gap-dirichlet-single-channel/reports/gap-dirichlet-single-routeA.md:25-35`, `65-75`). Route B correctly says the single quotient has not overtaken the paired route from current scope alone (`tasks/20260423-140208-attack-gap-dirichlet-single-channel/reports/gap-dirichlet-single-routeB.md:3-20`, `37-47`).

  Conditional on [a primitive-Dirichlet theorem identifying the exact realized single-channel scalar with the manuscript's `S(m)` slot]:

  - The paired route would no longer deserve to be called the conservative scalar-amplitude target for positivity **from that stronger scope alone**, because the main currently advertised advantage of the pair would collapse at the scalar-slot level.
  - Even then, the note would still have to keep separate the open denominator, whitening, remainder-dominance, and odd/transverse realization steps.

  Missing:

  - A primitive-Dirichlet theorem whose conclusion is the exact local value-channel scalar occupying the manuscript's `S(m)` slot in a single completed channel.
  - A proof that this scalar is real and nonnegative in the required local regime, not merely that an upstairs quotient kernel is termwise nonnegative.
  - Dirichlet analogues of denominator comparability, corrected whitening, remainder dominance, and the odd/transverse cancellation channel.

- Strongest objection

  The strongest objection is that the phrase `conservative scalar-amplitude target` may now overstate the paired route's advantage if readers interpret `single-channel source theorem` in the strengthened sense already isolated by Route A. Once exact slot-identification is granted, the pair is not obviously more conservative for scalar positivity itself; at that point it is only one remaining packaging option among others. So any note that says the pair is simply `the safer route` or `the conservative target` without the qualifier `from current scope alone` is vulnerable to overclaim.

- Needed for promotion

  - Replace any unscoped sentence of the form `the paired route is the conservative scalar-amplitude target` with `from current scope alone, the paired route remains the more conservative first target for realizing a real nonnegative scalar in the manuscript's exact S-slot`.
  - Replace any sentence suggesting `quotient positivity` or `unimodularity` solves the scalar-amplitude problem with the scoped wording: `conditional on a family-specific quotient/source theorem, the single completed channel gives universal upstairs one-zero kernel positivity; exact local S-slot realization is still missing`.
  - Weaken any sentence that treats the single route as only a `reality candidate`; the safe update is `conditional phase-channel candidate, with conditional upstairs kernel positivity, but not yet an exact local value-channel realization`.
  - Weaken any sentence implying the paired route remains conservative in every future scope. Safe version: `the paired route remains conservative from the present evidence base; this ranking can change if a single-channel theorem identifies the exact S-slot scalar`.
  - Honest verdict: the paired route still deserves the label `conservative scalar-amplitude target` only with explicit scope. The strongest safe Dirichlet note is a scoped ranking sentence that distinguishes exact `S(m)`-slot realization from phase and upstairs-kernel gains. The claims that need weakening are precisely the unscoped ones: any sentence equating quotient positivity with scalar-slot realization, any sentence calling the pair intrinsically safer, and any sentence leaving the single route at the weaker `mere reality candidate` level after the quotient-phase and universal-kernel notes.
