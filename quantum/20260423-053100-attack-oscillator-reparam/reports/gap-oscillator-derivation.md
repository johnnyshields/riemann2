# gap-oscillator-derivation

- **Claim**: For the oscillator ground-state family
  \[
  \psi_\omega(x)=\left(\frac{\omega}{\pi}\right)^{1/4}e^{-\omega x^2/2},\qquad \omega>0,
  \]
  with first jet pair \((|\psi\rangle,\partial|\psi\rangle)\), the two parameterizations \(\omega\) and \(r=\tfrac12\log\omega\) give different same-point and mixed blocks but the same bilaterally whitened two-point block. The same-point blocks are \(M_\omega(\omega)=\begin{pmatrix}1&0\\0&\frac1{8\omega^2}\end{pmatrix}\) and \(M_r(r)=\begin{pmatrix}1&0\\0&\frac12\end{pmatrix}\), so the nonconstancy in \(\omega\) is removed in \(r\). The mixed blocks are related by endpoint rescaling, and the whitened blocks coincide exactly in this example.
- **Status**: proved
- **Evidence**: Proved: from \(\partial_\omega\psi_\omega=\psi_\omega(\frac1{4\omega}-\frac{x^2}{2})\), one gets \(\langle\psi_\omega,\partial_\omega\psi_\omega\rangle=0\) and \(\|\partial_\omega\psi_\omega\|^2=\frac1{8\omega^2}\), hence \(M_\omega\). Proved: with \(r=\tfrac12\log\omega\), \(\partial_r\psi=2\omega\,\partial_\omega\psi\), so \(\|\partial_r\psi\|^2=\frac12\), hence \(M_r\). Proved: all mixed entries are derivatives of the closed overlap \(I(\omega_1,\omega_2)=\sqrt2\,(\omega_1\omega_2)^{1/4}/\sqrt{\omega_1+\omega_2}=\operatorname{sech}^{1/2}(\Delta_r)\), giving explicit \(N_\omega\) and \(N_r\). Proved: \(N_r=D_1N_\omega D_2\) with endpoint derivative rescalings, and in this benchmark the bilateral whitening cancels those rescalings exactly, so \(\Omega_r=\Omega_\omega\). Conditional: this is a statement about the raw first-jet benchmark built from \((|\psi\rangle,\partial|\psi\rangle)\). Missing: any proof that the surviving whitened block is canonical beyond this chosen first-jet construction and this reparameterization calculation.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:103-172`; `quantum/findings.md:30-39,54-58,62-67`; `quantum/20260423-051725-attack-jet-qgt-core/reports/gap-nonconstant-example.md:3-17`; `quantum/20260423-053100-attack-oscillator-reparam/notes/coordinator-brief.md:5-13`.
- **Dependencies**: Accept the benchmark rule that the first jet pair is \((|\psi(\lambda)\rangle,\partial_\lambda|\psi(\lambda)\rangle)\), equivalently the horizontal derivative here because \(\langle\psi,\partial_\lambda\psi\rangle=0\). Accept the paper-style bilateral whitening. Use only the reparameterization \(r=\tfrac12\log\omega\).
- **Strongest objection**: The exact coincidence \(\Omega_r=\Omega_\omega\) is proved here only because the change of parameter acts by simple scalar Jacobian factors on the one-dimensional derivative channel, and those factors are exactly canceled by the bilateral whitening in this example. This does not by itself prove a general canonical two-point quantum invariant.
- **Needed for promotion**: Prove the general reparameterization law for the chosen jet model beyond this one-dimensional first-jet example, or isolate the smallest exact invariant class actually justified.
