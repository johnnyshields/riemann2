# gap-frame-criterion

- **Claim**: Let \(S_-,S_+\subset\mathcal H\) be the canonical endpoint subspaces in a common ambient Hilbert space, let \(C:=\Pi_-|_{S_+}:S_+\to S_-\) be the canonical cross-contraction, and let the nonzero-overlap sector mean the support of \(|C|=(C^*C)^{1/2}\). Then a canonical orthonormal principal frame on the nonzero-overlap sector exists from the subspace pair alone, in the natural unitary-equivariant sense, **iff** every nonzero principal value is simple. In that case the frame is canonical up to independent simultaneous phase/sign changes on each matched pair \((u_i,v_i)\), with diagonal comparison matrix \(\operatorname{diag}(\cos\theta_i)\). If any nonzero principal value has multiplicity \(m\ge2\), no such canonical orthonormal frame exists on that degenerate nonzero block; the strongest exact canonical object there is only the block partial isometry from the polar decomposition of \(C\).
- **Status**: proved
- **Evidence**: Proved: the simple-spectrum corollary gives canonical matched principal vectors on each simple nonzero eigenspace of the canonical compression. The degenerate-sector no-go shows that repeated nonzero principal values leave an unavoidable internal \(U(m)\) symmetry, so no canonical orthonormal basis can be extracted from the pair alone on that block. Together these give the stated iff criterion on the nonzero-overlap sector.
