# gap-projector-operator

- **Claim**: The cleanest basis-free canonical operator is the projector product. Let \(P_\pm\) be the orthogonal projections onto the canonical ambient osculating subspaces \(A_r(x_\pm)\subset \mathcal H\). Then the exact operator-level invariant is
  \[
  K_-:=P_-P_+P_- \quad \text{on } A_r(x_-),
  \qquad
  K_+:=P_+P_-P_+ \quad \text{on } A_r(x_+).
  \]
  If \(\theta_1,\dots,\theta_m\) are the principal angles between \(A_r(x_-)\) and \(A_r(x_+)\), where \(m=\min(\dim A_r(x_-),\dim A_r(x_+))\), then the nonzero spectrum of \(K_\pm\) is exactly
  \[
  \{\cos^2\theta_1,\dots,\cos^2\theta_m\}
  \]
  with multiplicities, and the singular values of \(P_-P_+\) are exactly \(\cos\theta_j\). Since the subspaces \(A_r(x)\) are already proved canonical, these spectra are canonical transport-free two-point invariants. This is the strongest exact theorem presently justified, in both the one-parameter and multiparameter ambient-subspace settings. If one wants the principal-angle data in operator form, projector products are the right object.
- **Status**: proved
- **Evidence**: Proved: the one-parameter report proves that \(A_r(\lambda)\) is exactly invariant under local phase gauge and under \(C^r\) reparameterization with nonzero derivative, so the pair \((A_r(\lambda_-),A_r(\lambda_+))\) is canonical in a fixed ambient Hilbert space. Proved: the multiparameter report proves the same statement with \(A_r(x)\) built from all mixed partials of total order \(\le r\), invariant under gauge and chart change. Proved: for any pair of finite-dimensional subspaces of a Hilbert space, the principal angles are encoded by orthogonal-projector products: \(P_-P_+P_-\) on \(A_r(x_-)\) and \(P_+P_-P_+\) on \(A_r(x_+)\) have eigenvalues \(\cos^2\theta_j\), while \(P_-P_+\) has singular values \(\cos\theta_j\). So once the subspaces are canonical, these spectra are canonical too. Conditional: choosing bases inside \(A_r(x_\pm)\) and writing cross-Gram or whitened matrices gives representatives whose entries depend on frame and, in the earlier transport language, on transport choices; only the projector-product spectrum is forced canonically by the proved subspace theorem. Missing: no stronger exact theorem identifies a canonical matrix-valued two-point object whose entries carry more invariant information than the principal-angle data.

- **Dependencies**: A fixed ambient Hilbert space \(\mathcal H\); the already-proved canonicity of \(A_r(x)\) at one parameter or for mixed partials in several parameters; orthogonal projections onto closed finite-dimensional subspaces; standard Hilbert-space principal-angle theory equating \(\sigma(P_-P_+)\) with \((\cos\theta_j)\) and \(\operatorname{spec}(P_-P_+P_-|_{A_r(x_-)})\) with \((\cos^2\theta_j)\).
- **Strongest objection**: This does not improve the invariant content beyond principal angles. It repackages the same data operator-theoretically, but it does not rescue a canonical raw cross-Gram or bilaterally whitened matrix. Also, it requires both endpoint subspaces to sit in the same ambient \(\mathcal H\); without a common ambient realization, the projector products are not defined.
- **Needed for promotion**: State the theorem directly in projector language: the orthogonal projections \(P_\pm\) onto the canonical ambient osculating subspaces define canonical compression operators \(P_-P_+P_-\) and \(P_+P_-P_+\), whose spectra are exactly the squared cosines of the principal angles. Present this as the strongest exact operator-level invariant now available, and explicitly avoid any stronger matrix-entry canonicity claim.
