# verifier-adversarial

- **Claim**: The strongest exact multiparameter extension safe to add is the ambient-subspace theorem and nothing stronger: let \(U\subset \mathbb R^d\) be a coordinate chart, let \(\lambda\mapsto |\psi(\lambda)\rangle\) be a normalized \(C^r\) local lift into a fixed ambient Hilbert space, and define for each \(\lambda\)
  \[
  \mathcal A_r(\lambda):=\operatorname{span}\{\psi(\lambda),\partial^\alpha_\lambda\psi(\lambda):1\le |\alpha|\le r\},
  \]
  equivalently
  \[
  O_r(\lambda):=\operatorname{span}\{P(\lambda)\partial^\alpha_\lambda\psi(\lambda):1\le |\alpha|\le r\},\qquad P=I-|\psi\rangle\langle\psi|.
  \]
  Then \(\mathcal A_r(\lambda)\) and \(O_r(\lambda)\) are invariant under local phase gauge and under smooth local changes of chart with invertible Jacobian. Hence, when comparing two parameter points inside the same fixed ambient Hilbert space and the endpoint ranks match, the principal angles between the endpoint subspaces are canonical ambient invariants. No multiparameter matrix-valued transport-free invariant is proved from the present scope alone.
- **Status**: proved
- **Evidence**: Proved: the one-parameter record already establishes the mechanism that matters: higher jets are not individually canonical because gauge changes and reparameterizations mix order \(k\) with lower orders by invertible triangular laws, but the span survives. In several parameters the same repair survives if one uses the full set of mixed partials up to order \(r\): under \(|\psi\rangle\mapsto e^{i\chi}|\psi\rangle\), Leibniz expands each \(\partial^\alpha(e^{i\chi}\psi)\) as \(e^{i\chi}\) times \(\partial^\alpha\psi\) plus lower-order derivatives, so the span is unchanged; after projection by \(P\), only lower-order projected jets remain. Under a chart change \(\mu\mapsto \lambda(\mu)\), multivariate chain rule / Faà di Bruno writes each \(\partial^\alpha_\mu\psi\) as a linear combination of \(\partial^\beta_\lambda\psi\) with \(1\le |\beta|\le |\alpha|\), and the top-order block is the symmetric-power action of the Jacobian, hence invertible when the Jacobian is invertible. Therefore the full order-\(\le r\) ambient osculating span is chart-invariant. Conditional: once those endpoint subspaces are canonical and lie in a common ambient Hilbert space, their principal angles are canonical two-point invariants. Missing: any proof that a bilaterally whitened multiparameter jet matrix survives these non-unitary jet-coordinate changes, any canonical frame selection on the endpoint subspaces, and any transport-independent multiparameter matrix theorem.

- **Dependencies**: Fixed ambient Hilbert space; normalized \(C^r\) local lift; local phase \(\chi\in C^r\); local chart changes of class \(C^r\) with invertible Jacobian; inclusion of all mixed partials up to total order \(r\); projector identity \(P\psi=0\); multivariate Leibniz rule and Faà di Bruno; standard linear algebra for principal angles.
- **Strongest objection**: This does not rescue the earlier matrix story. The safe multiparameter object is an ambient osculating subspace built from all mixed partials up to order \(r\), not a chart-dependent ordered jet list and not a transport-free bilaterally whitened matrix. From present scope alone, any stronger claim about canonical multiparameter matrix entries, transport-free whitening, or a preferred multiparameter higher-jet frame would overclaim.
- **Needed for promotion**: State the theorem exactly at the ambient-subspace level, with “all mixed partials of total order \(\le r\)” explicit. If a two-point corollary is wanted, state only principal angles between endpoint subspaces in a fixed ambient Hilbert space. Do not promote any multiparameter whitened-matrix invariant unless one proves a canonical higher-jet/frame/transport package or proves that the matrix depends only on the subspaces.
