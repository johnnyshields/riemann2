# verifier-adversarial

- **Claim**: The strongest exact addendum currently safe for `quantum/paper/jet_gram_quantum_note.md` is this narrow first-order statement: for a \(C^1\) one-parameter pure-state ray curve with nonvanishing first horizontal jets at the two endpoints, and after choosing a gauge-covariant two-point transport rule \(U_{-\leftarrow +}\), the bilaterally whitened first-horizontal-jet block is independent of local lift and invariant under orientation-preserving reparameterization. For an orientation-reversing reparameterization \(\mu=\phi(\lambda)\), with endpoint labels kept fixed by the original ordered pair \((\lambda_-,\lambda_+)\), the exact replacement for equality is
  \[
  \Omega_\mu = E\,\Omega_\lambda\,E,\qquad E=\mathrm{diag}(1,-1).
  \]
  So reversal flips only the derivative channel; it does not give literal matrix equality. What cannot be added, from the present scope alone, is any claim that the matrix is transport-independent, endpoint-only, or ambiguous only by a sign/global phase.
- **Status**: proved
- **Evidence**: Proved: the one-parameter first-horizontal-jet same-point block is diagonal because \(\langle \psi|h_\lambda\rangle=0\), and under reparameterization \(h_\mu=(d\lambda/d\mu)h_\lambda\); for \(d\lambda/d\mu>0\) the endpoint rescalings cancel exactly under bilateral whitening, giving \(\Omega_\mu=\Omega_\lambda\). Proved: the same first-order algebra also gives the orientation-reversing law. If \(d\lambda/d\mu<0\), then the derivative channel picks up one sign at each endpoint, so \(\Omega_\mu=E\Omega_\lambda E\). Proved: with a gauge-covariant transport rule, the block is independent of the local lift, so the surviving object is a ray-curve-with-transport invariant. Conditional: if one weakens from the full matrix to transported horizontal-line data, then orientation-insensitive quantities such as singular values or principal-angle data remain plausible honest invariants. Missing: any proof that different admissible transport rules give the same matrix, any proof that the ambiguity reduces universally to a sign or global phase, and any proof that the matrix is determined by endpoints alone rather than the chosen curve/transport.
