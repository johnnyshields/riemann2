# gap-berry-pancharatnam

- **Claim**: For the one-parameter first-horizontal-jet block with fixed endpoint order and nonzero endpoint overlap \(\langle \psi_-|\psi_+\rangle\neq 0\), Berry and Pancharatnam transports lie in the same scalar-phase transport class: each acts on the endpoint \(+\) first-jet pair by one common phase \(e^{i\theta}\), so after whitening the resulting blocks satisfy \(\Omega_{\mathrm P}=e^{i\delta}\Omega_{\mathrm B}\) for some unit scalar \(e^{i\delta}\). Hence, within this restricted class, Berry versus Pancharatnam changes the whitened first-horizontal-jet block only by a global phase, not entrywise.
- **Status**: proved
- **Evidence**: Proved: the first-jet formula reduces the whitened block to overlaps of the orthonormalized endpoint frame \((\psi_\pm,e_\pm)\), where \(e_\pm:=h_{\lambda_\pm}/\|h_{\lambda_\pm}\|\). For any scalar-phase transport, both \(\psi_+\) and \(e_+\) acquire the same phase, so every matrix entry acquires that phase. Berry transport and Pancharatnam transport are both of this form: Berry fixes a parallel-transport phase along the oriented curve segment, while Pancharatnam fixes the unique extra endpoint phase making the transported overlap with \(\psi_-\) positive real. When \(\langle\psi_-|\psi_+\rangle\neq 0\), that extra phase exists and is unique, so the two transports differ by a single scalar phase, hence the two whitened blocks differ by the same global phase. Missing: zero-overlap cases and any proof that the scalar-phase class is canonical rather than imposed.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:107-205`; `quantum/findings.md:35-49`; `quantum/findings.md:91-98`; `quantum/teams/20260423-130653-attack-first-jet-remaining/reports/gap-transport-dependence.md:3-17`; `quantum/teams/20260423-131101-attack-transport-class/notes/coordinator-brief.md:6-14`.
- **Dependencies**: One-parameter \(C^1\) pure-state ray curve with fixed oriented segment; first horizontal jet; nonvanishing endpoint horizontal norms; nonzero endpoint overlap for Pancharatnam transport; standard definitions of Berry and Pancharatnam phase as scalar rephasings of the endpoint lift on that curve segment.
- **Strongest objection**: The conclusion is only as strong as the restriction to the scalar-phase transport class. If one allows more general gauge-covariant transports on the \(2\)-dimensional span \(\operatorname{span}\{\psi_+,e_+\}\), the earlier counterexample applies and the ambiguity can be entrywise, not global.
- **Needed for promotion**: State the transport class explicitly in the note: admissible transports for this first-jet theorem are scalar rephasings of the endpoint first-jet pair, with Pancharatnam available only when endpoint overlap is nonzero. Then the safe promoted statement is: within that class, Berry and Pancharatnam give the same whitened first-horizontal-jet block up to one global phase.
