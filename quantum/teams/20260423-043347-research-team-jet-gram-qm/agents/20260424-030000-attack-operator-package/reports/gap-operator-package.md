# gap-operator-package

- **Claim**: Let \(S_-,S_+\subset \mathcal H\) be a finite-dimensional canonical pair of endpoint subspaces in a common ambient Hilbert space, with orthogonal projectors \(\Pi_\pm\). Then the complete exact canonical operator package determined by the pair is:
  \[
  C:=\Pi_-\!\big|_{S_+}:S_+\to S_-,\qquad
  K_+:=C^*C=\Pi_+\Pi_-\Pi_+\!\big|_{S_+},\qquad
  K_-:=CC^*=\Pi_-\Pi_+\Pi_-\!\big|_{S_-}.
  \]
  These are canonical basis-free operators; \(C\) is a contraction; \(K_\pm\) are canonical positive compressions whose nonzero spectra are exactly the squared principal-angle cosines \(\{\cos^2\theta_i\}\) with multiplicities. The polar decomposition
  \[
  C=V|C|,\qquad |C|=K_+^{1/2},
  \]
  gives a canonical partial isometry \(V\) from the nonzero-overlap sector of \(S_+\) onto that of \(S_-\), intertwining the matching nonzero spectral subspaces of \(K_+\) and \(K_-\). If the nonzero spectrum is simple, this upgrades to canonical matched principal vectors up to independent phase/sign, hence a canonical diagonal comparison matrix with entries \(\cos\theta_i\). If a nonzero principal value has multiplicity \(m\ge 2\), no canonical orthonormal principal frame is determined by the pair alone on that block; the sharp canonical object there is only the block partial isometry \(V_\sigma\). This is the strongest exact finite-dimensional operator-level synthesis presently justified. It adds no invariant content beyond principal-angle / projector-spectrum data and does not yield a canonical full entrywise matrix package.
- **Status**: proved
- **Evidence**: Proved: the projector-product theorem gives canonical operators \(K_\pm\) for a canonical subspace pair, with nonzero spectra equal to \(\cos^2\theta_i\), so the positive compression package is exact and basis-free. Proved: the cross-contraction \(C=\Pi_-|_{S_+}\) is canonical, satisfies \(C^*=\Pi_+|_{S_-}\), \(C^*C=K_+\), \(CC^*=K_-\), and therefore has a canonical polar decomposition \(C=V|C|\); \(V\) is the strongest exact operator-level transport on the nonzero-overlap sector. Proved: when the nonzero spectrum is simple, each nonzero eigenspace is one-dimensional, so the corresponding principal vectors are canonical up to phase/sign and give a diagonal normal form. Proved: when a nonzero principal value is repeated, the associated block carries an unavoidable internal \(U(m)\) freedom, so no canonical orthonormal principal frame can be recovered from the subspace pair alone; only the degenerate spectral subspaces and the block partial isometry remain canonical.

- **Dependencies**: Finite-dimensional subspace pair \(S_\pm\subset\mathcal H\) in a common ambient Hilbert space; canonicity of the endpoint subspaces from the upstream subspace theorem; orthogonal projectors \(\Pi_\pm\); standard finite-dimensional principal-angle theory; spectral theorem for positive self-adjoint operators; polar decomposition for operators between Hilbert spaces; unitary-equivariance as the meaning of “canonical from the pair alone” in the degenerate-sector no-go.
- **Strongest objection**: The synthesis is exact only at the basis-free operator level. It does not produce a canonical raw cross-Gram / whitened matrix, and the simple-spectrum frame statement is merely a corollary under a nondegeneracy hypothesis. In repeated nonzero spectral blocks, any frame choice needs extra symmetry-breaking structure.
- **Needed for promotion**: Promote only as a theorem explicitly subordinate to the canonical subspace-pair theorem: first define \(C\), \(K_\pm\), and \(V\); then state the simple-spectrum diagonal-frame result as a corollary; then state the repeated-spectrum no-go as the sharp converse.
