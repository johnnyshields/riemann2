# gap-grassmannian-metrics

- **Claim**: Let \(x \mapsto A_r(x)\) or \(x \mapsto O_r(x)\) be the canonical ambient or value-channel-free osculating subspace family in a fixed ambient Hilbert space \(\mathcal H\). On any locus where the subspace dimension is constant, this defines an exact map into a Grassmannian \(\mathrm{Gr}(k,\mathcal H)\). The strongest exact metric statement presently justified is: the principal angles \(\theta_1,\dots,\theta_k\) between the endpoint subspaces are canonical, hence every Grassmannian distance that is an exact function of those principal angles is canonical as well. In particular,
  \[
  d_{\mathrm{proj}}(E,F)=\|\Pi_E-\Pi_F\|_{\mathrm{op}}=\max_i \sin\theta_i,
  \]
  and, in the finite-rank Hilbert-Schmidt setting,
  \[
  d_{\mathrm{ch}}(E,F)=2^{-1/2}\|\Pi_E-\Pi_F\|_{\mathrm{HS}}=\Big(\sum_i \sin^2\theta_i\Big)^{1/2}.
  \]
  Equivalently, the compression \(\Pi_E\Pi_F\Pi_E|_E\) has eigenvalues \(\cos^2\theta_i\), so the Grassmannian metric data is exactly the same data already encoded by the canonical projector product. This is exact for both \(A_r\) and \(O_r\); for \(A_r\) it is already in the note’s strongest transport-free theorem, while for \(O_r\) it is exact on the same ambient-subspace footing whenever the endpoint ranks match.
- **Status**: proved
- **Evidence**: Proved: the note and supporting reports establish that \(A_r(x)\) and \(O_r(x)\) are exactly invariant under local phase gauge and under reparameterization/chart change, so on a constant-rank locus they are honest maps into a Grassmannian rather than coordinate artifacts. Proved: once two endpoint subspaces in a common ambient \(\mathcal H\) are canonical, their principal angles are canonical. Proved: the projector compression \(\Pi_-\Pi_+\Pi_-|_{E_-}\) has spectrum \(\{\cos^2\theta_i\}\), so any exact Grassmannian metric expressible through the \(\theta_i\) is immediately canonical; the projection/chordal formulas above are standard exact identities for Grassmannians. Conditional: if one wants a specifically Riemannian infinitesimal metric along the parameter manifold \(x\), one must still assume constant rank and enough regularity of the projector-valued map \(x\mapsto \Pi_{A_r(x)}\) or \(x\mapsto \Pi_{O_r(x)}\). Missing: no exact transport-free matrix-entry theorem beyond this subspace/principal-angle/projector content, and no global statement across rank-jump points.

- **Dependencies**: Fixed ambient Hilbert space \(\mathcal H\); canonical subspace family \(A_r\) or \(O_r\); comparison only on a locus of constant dimension or between endpoint pairs with matching ranks; standard principal-angle theory for finite-dimensional subspaces; orthogonal projectors \(\Pi_E,\Pi_F\); for the Hilbert-Schmidt chordal formula, finite rank.
- **Strongest objection**: This is a Grassmannian metric statement only at the subspace level. It does not recover a canonical bilaterally whitened matrix, it does not decide whether \(A_r\) or \(O_r\) is the uniquely preferred object, and it is not a single global metric theory across points where the rank changes.
