# verifier-adversarial

- **Claim**: For the oscillator ground-state family, once the reparameterization issue is taken seriously, the honest surviving content is narrow. In the change `\omega \leftrightarrow r=\tfrac12\log\omega`, the nonconstant same-point first-jet Gram entry is a coordinate artifact, while the first-jet ray-subspace `\mathrm{span}\{\psi,\partial\psi\}` and the bilaterally whitened first-jet two-point block survive unchanged in this specific one-parameter first-order comparison. What this example buys the future draft is therefore a benchmark that cleanly separates parameterization-sensitive matrix behavior from subspace-level data; it does not, from this scope alone, justify a general canonical matrix-valued higher-jet invariant.
- **Status**: proved
- **Evidence**: Proved: the note already flags the key warning that `M(\omega)=\mathrm{diag}(1,1/(8\omega^2))` is nonconstant in `\omega` but can be normalized away in `r`, so that nonconstancy by itself is not intrinsic. Proved: under `r=\tfrac12\log\omega`, one has `\partial_r\psi_\omega=\omega\,\partial_\omega\psi_\omega`, so the first-jet line inside the horizontal tangent space is unchanged; only its coordinate scale changes. Proved: the same-point Gram block correspondingly changes from `\mathrm{diag}(1,1/(8\omega^2))` to `\mathrm{diag}(1,1/8)`, so that entry is parameterization-sensitive matrix data. Proved: for the two-point first-jet construction built from `(|\psi\rangle,\partial_\lambda|\psi\rangle)`, the mixed block in `r` is obtained from the mixed block in `\omega` by endpoint rescalings of the derivative row/column, and the bilateral whitening cancels those rescalings exactly; hence the whitened `2\times2` block is the same in `\omega` and `r` for this family. Conditional: this surviving matrix equality is only for this one-parameter first-order reparameterization, where the jet change is a scalar rescaling with no higher-jet triangular mixing. Missing: any argument that the same stability persists for higher jets, multiparameter families, nontrivial transport/gauge issues, or arbitrary frame choices.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:103-172`; `quantum/findings.md:35-39`; `quantum/findings.md:54-58`; `quantum/teams/20260423-051725-attack-jet-qgt-core/reports/gap-nonconstant-example.md:3-17`; `quantum/teams/20260423-051725-attack-jet-qgt-core/reports/gap-matrix-vs-subspace.md:3-9`; `quantum/teams/20260423-053100-attack-oscillator-reparam/notes/coordinator-brief.md:5-13`.
- **Dependencies**: The first-jet model is exactly `(|\psi\rangle,(I-|\psi\rangle\langle\psi|)\partial_\lambda|\psi\rangle)`; for this real normalized family the horizontal projection does not change the derivative. The comparison uses only the parameter change `r=\tfrac12\log\omega` and the bilateral whitening rule already adopted in the note.
- **Strongest objection**: The surviving whitened-matrix equality may tempt an overclaim, but it is special to the first-jet, one-parameter oscillator benchmark. Here reparameterization acts by a scalar on the derivative direction, and the family has trivial Berry structure. From this scope alone, it does not settle the general matrix-vs-subspace question.
- **Needed for promotion**: To promote this into draft-level positive doctrine, prove a general theorem showing when first-jet bilateral whitening is reparameterization invariant, or else keep the oscillator lesson as a warning example plus benchmark.
