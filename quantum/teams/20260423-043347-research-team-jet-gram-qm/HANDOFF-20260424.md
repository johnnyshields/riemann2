## Current High-Level Conclusion

The central conclusion of the track is now substantially sharper than when it started:

1. The strongest robust output is not a canonical two-point matrix.
2. The strongest robust output is a canonical **subspace/operator package**.
3. First-order matrix structure survives in a very limited and exact way.
4. Higher-order matrix structure fails in general, but higher-order subspace data survives exactly.
5. The surviving two-point data is richer than overlap alone on a large and now fairly explicit benchmark class.
6. In finite dimension, the principal-angle/operator package is complete for classifying subspace pairs up to ambient unitary equivalence.

What remains open is not “does anything canonical survive?” It does. What remains open is whether there is any **strictly stronger canonical matrix/frame package** beyond the operator/subspace data without extra frame-selecting structure.

## Exact Confirmed Results

### 1. Source-side translation

Safe source-side label:
- `whitened local jet cross-Gram block`

This is the safest quantum-side name for the source paper’s local object.

### 2. First-order same-point theorem

For a normalized pure-state family `|psi(lambda)>`, with

\[
P=I-|\psi\rangle\langle\psi|,
\qquad
|D_a\psi\rangle=P\,\partial_a|\psi\rangle,
\]

the same-point horizontal Gram matrix

\[
Q_{ab}=\langle D_a\psi\mid D_b\psi\rangle
=\langle\partial_a\psi\mid(I-|\psi\rangle\langle\psi|)\mid\partial_b\psi\rangle
\]

is exactly the pure-state QGT.

Consequences:

\[
g_{ab}=\Re Q_{ab},
\qquad
F_{ab}=2\Im Q_{ab},
\qquad
I_{ab}=4\Re Q_{ab}.
\]

This is the clean first-order bridge.

### 3. First-order two-point theorem

For one-parameter pure-state curves and a chosen two-point transport rule,
the bilaterally whitened first-horizontal-jet block is exactly invariant under orientation-preserving reparameterization.

Under orientation reversal:

\[
\Omega \mapsto E\Omega E,
\qquad E=\operatorname{diag}(1,-1).
\]

So only the derivative channel flips sign.

### 4. Gauge upgrade at first order

With a gauge-covariant transport rule, the first-horizontal-jet block descends from a fixed-lift construction to a ray-curve construction.

Important limitation:
- This is still not a transport-independent endpoint matrix theorem.

### 5. Scalar-phase transport class

If the transport is restricted to the scalar-phase class on the endpoint first-jet pair, then the first-horizontal-jet block changes only by a global phase.

In that reduced class:
- Berry and Pancharatnam transport differ only by a global phase when endpoint overlap is nonzero.

### 6. First-order singular-value theorem

For fixed gauge-covariant transport, the singular values of the first-horizontal-jet block are:
- lift-invariant
- invariant under orientation-preserving reparameterization
- invariant under orientation reversal

In the scalar-phase class they are also transport-class invariant.

### 7. Higher-jet no-go at matrix level

For raw projected jets

\[
j_k=(I-|\psi\rangle\langle\psi|)\partial_\lambda^k\psi,
\]

higher-order ordered jets transform by upper-triangular laws under gauge change and reparameterization.

So from second order onward:
- raw jet matrices are not canonical
- raw Gram/whitened matrices are not canonical

### 8. Higher-jet subspace theorem

What survives exactly is

\[
O_r=\mathrm{span}\{j_1,\dots,j_r\}.
\]

This is exact under one-parameter gauge and reparameterization.

### 9. One-parameter higher-order covariant jets

Define

\[
a(t)=\langle\psi,\partial_t\psi\rangle,
\qquad
\nabla_t\xi=P_t(\partial_t\xi-a(t)\xi).
\]

Then the recursively defined jets

\[
h_{[1]}=\nabla_t\psi,
\qquad
h_{[k+1]}=\nabla_t h_{[k]}
\]

are gauge-covariant and span the same `O_r` spaces as the raw projected jets.

So the one-parameter problem is no longer “find any covariant hierarchy.”
That part is closed.

### 10. Multiparameter ambient theorem

For all mixed partials up to total order `<= r`, the ambient subspace

\[
A_r(x)=\mathrm{span}\{\psi(x),\partial^\alpha\psi(x):1\le |\alpha|\le r\}
\]

is chart- and gauge-invariant in a fixed ambient Hilbert space.

The projected version

\[
O_r(x)=\mathrm{span}\{P\partial^\alpha\psi(x):1\le |\alpha|\le r\}
\]

is also invariant.

Therefore the principal angles between endpoint `A_r` or `O_r` spaces are transport-free canonical invariants.

### 11. Multiparameter covariant jets

With Berry components

\[
a_a=\langle\psi,\partial_a\psi\rangle,
\qquad
\nabla_a\xi=P(\partial_a\xi-a_a\xi),
\]

ordered iterated covariant derivatives span the same canonical `O_r` spaces as the raw mixed partials.

This closes the subspace-level multiparameter covariant-hierarchy problem.

### 12. Symmetrized multiparameter jets

The multiparameter covariant-jet hierarchy can be reduced further:
- ordered covariant jets and symmetrized covariant jets generate the same `O_r` filtration.

So ordering is no longer the real obstruction.
The real remaining obstruction is canonical matrix/frame data.

### 13. Ambient operator package

For a canonical subspace pair `(S_-, S_+)`, the complete exact operator package is:

\[
C=\Pi_-|_{S_+},
\qquad
K_+=C^*C,
\qquad
K_-=CC^*,
\qquad
C=V|C|.
\]

Interpretation:
- `C` is the canonical cross-contraction
- `K_±` are the canonical positive compressions
- `V` is the canonical polar partial isometry on the nonzero-overlap sector

The nonzero spectra of `K_±` are exactly the squared principal-angle cosines.

### 14. Simple-spectrum positive result

If the nonzero principal spectrum is simple, then canonical principal vectors exist up to independent phase/sign, and the comparison matrix can be canonically diagonalized on that sector.

### 15. Degenerate-sector negative result

If a nonzero principal value has multiplicity `m >= 2`, there is no canonical orthonormal principal frame from the subspace pair alone in the unitary-equivariant sense.

What survives canonically there is only:
- the degenerate principal subspace
- the block partial isometry from the polar decomposition

### 16. Exact iff criterion for canonical principal frames

On the nonzero-overlap sector:
- canonical principal frames exist from the pair alone iff the nonzero spectrum is simple.

### 17. Completeness in finite dimension

For finite-dimensional pairs, the principal-angle/operator package is complete up to ambient unitary equivalence.

That is:
- endpoint dimensions
- ambient dimension
- principal-angle multiset with multiplicities

classify the pair up to unitary equivalence.

This is now backed internally by an explicit CS block normal form in the archive.

## Exact Dynamical Specializations

### 18. Exact unitary/Krylov theorem

For `psi(t)=e^{-itH} psi_0`:

\[
A_r(t)=e^{-itH}\,\mathrm{span}\{\psi_0,H\psi_0,\dots,H^r\psi_0\}.
\]

So `A_r` is exactly a transported Krylov space.

Also:

\[
O_r(t)=A_r(t)\cap \psi(t)^\perp
=e^{-itH}(K_r(H,\psi_0)\cap\psi_0^\perp).
\]

So `O_r` is exactly the value-channel-removed / compressed Krylov object.

### 19. Exact propagator-compression theorem

Along exact unitary orbits, the principal-angle data between endpoint subspaces is exactly the singular-value/spectral data of the propagator compressed to the initial Krylov or compressed-Krylov space.

### 20. Finite-cyclic rank growth and saturation

If the cyclic space has finite dimension `m`, then:

\[
\dim A_r = \min(r+1,m),
\qquad
\dim O_r = \min(r,m-1).
\]

For `r >= m-1`, the exact two-point invariants stabilize.

### 21. Finite-cyclic Jacobi/Lanczos normal form

In the finite-cyclic case, the spectral measure of `(H, psi_0)` gives a canonical orthogonal-polynomial basis and a finite Jacobi matrix `J` such that the ambient two-point package is exactly represented by `e^{-iΔJ}`.

Important limitation:
- this is an exact specialization of the already-known unitary package, not a new general canonical matrix theorem.

### 22. Local projector metric and leakage

For a smooth constant-rank projector path `Π(x)`:

\[
g^{\Pi}_{ab} = \frac12 \operatorname{Tr}((\partial_a\Pi)(\partial_b\Pi)).
\]

In rank one this recovers exactly `Re Q`.

Along exact unitary subspace orbits:

\[
\dot\Pi = -i[H,\Pi],
\qquad
\frac12\|\dot\Pi\|_{HS}^2
=\frac12\|[H,\Pi]\|_{HS}^2
=\|(I-\Pi)H\Pi\|_{HS}^2.
\]

### 23. Kato transport theorem

Every smooth projector path has an exact pathwise unitary realization:

\[
H_K(t)=i[\dot\Pi(t),\Pi(t)],
\qquad
\Pi(t)=U_K(t)\Pi(0)U_K(t)^*.
\]

This gives a canonical pathwise dynamical model, but it is not endpoint-canonical.

### 24. Kato curvature obstruction

In several parameters, the exact curvature is

\[
R^K = \Pi(d\Pi \wedge d\Pi)\Pi.
\]

Nonzero curvature forces local holonomy, so endpoint-independent Kato transport fails whenever this curvature is nonzero.

Rank-one reduction:

\[
R^K = iF\,\Pi,
\]

so the obstruction is exactly Berry curvature.

## Benchmark Families

### 25. Oscillator benchmark

Family:

\[
\psi_\omega(x)=\left(\frac{\omega}{\pi}\right)^{1/4}e^{-\omega x^2/2}.
\]

Use:
- exact reparameterization sanity check.

Confirmed:
- raw first-jet matrices change under `ω <-> r = (1/2) log ω`,
- the bilaterally whitened first-jet block stays unchanged.

Negative:
- singular values collapse to a function of overlap alone.

### 26. Qutrit polynomial benchmark

Family:

\[
|\psi(t)\rangle=\frac{(1,t,t^2)^T}{\sqrt{1+t^2+t^4}}.
\]

Confirmed:
- ambient `A_1` side is richer than overlap alone,
- value-channel-free `O_1` side is also richer than overlap alone,
- explicit exact equal-overlap witnesses exist on both sides.

### 27. Quartit cubic benchmark

Family:

\[
|\psi(t)\rangle=\frac{(1,t,t^2,t^3)^T}{\sqrt{1+t^2+t^4+t^6}}.
\]

Confirmed:
- ambient `A_2` side has exact singular values `(1,1,σ_3)`,
- value-channel-free `O_2` side has an explicit exact `2x2` complement matrix,
- `O_2` is strictly finer than `A_2` on this benchmark.

### 28. Veronese family

Family:

\[
v_n(t)=(1,t,\dots,t^n).
\]

Ambient side:
- exact order-`n-1` theorem,
- codimension-1 hyperplane normal from coefficients of `(x-t)^n`,
- nontrivial cosine is a normalized binomial-square polynomial in `uv`.

Value-channel-free side:
- exact order-`n-1` codimension-2 complement theorem,
- explicit `2x2` complement matrix,
- generic richer-than-overlap behavior for all `n >= 2`.

Lower-order side:
- exact complement-basis theorem for `A_r^` and now also for `O_r^` via the direct-sum decomposition,
- but no equally sharp all-`(n,r)` spectral formula below top order.

### 29. Non-Veronese spherical twist family

Base curve:

\[
\psi(t)=(\cos t\cos 2t,\cos t\sin 2t,\sin t).
\]

Exact witness:
- `(0, π/4)` and `(0, π/2)` have equal overlap `0` but different `A_1` and `O_1` angle data.

Family version:

\[
\psi_k(t)=(\cos t\cos kt,\cos t\sin kt,\sin t),\qquad k\ge2.
\]

This family also has exact equal-overlap witnesses for every integer `k >= 2`.

Meaning:
- richer-than-overlap behavior is not confined to polynomial examples.

## What Is Still Only Partial Or Speculative

These are not false; they just are not promoted into the main note at full strength.

1. Smooth pathwise principal frames on simple-spectrum loci.
   The static simple-spectrum theorem is exact; the smooth pathwise refinement is only archived as a plausible regularity upgrade.

2. Smooth pathwise polar partial isometries on constant-rank loci.
   Also archived as a likely regularity theorem, but not promoted.

3. Grassmannian metric packaging.
   Exact but mostly repackaging of principal angles/projector spectra, so not promoted heavily.

4. Truncated moment-matrix interpretation.
   Exact in the unitary/Krylov specialization, but mostly basis-dependent repackaging, so not promoted heavily.

5. Lower-order all-`(n,r)` Veronese principal-angle formulas.
   Complement-basis theorems exist, but no equally sharp general spectral formulas are currently in hand.

6. Generic local independence beyond benchmarks.
   There is a correct Jacobian criterion, but it is not strong enough to promote as a general structural theorem yet.

## Main Remaining Open Problems

1. Canonical matrix/frame package beyond subspace data.
   This is still the main unresolved issue.
   The current strongest exact package is the operator package `(C, K_±, V)` plus the simple-spectrum and no-go corollaries.

2. Multiparameter matrix canonicity.
   We now have exact multiparameter covariant jets and symmetrized covariant jets at the subspace level.
   The remaining question is whether any canonical ordered matrix package exists beyond that.

3. `A_r` versus `O_r` outside the benchmark families.
   We now know their exact relation very well, including the quotient interpretation, but we do not yet have a universal criterion for which is “the” right object in every context.

4. Better dynamical / experimental models.
   We have exact unitary/Krylov, exact projector tests, exact Kato transport, exact leakage formulas.
   We do not yet have a more compelling physically natural model that clearly singles out the subspace package in an externally meaningful way.

5. Non-benchmark genericity.
   We now have polynomial and non-polynomial benchmark families, but not yet a strong theorem saying richer-than-overlap behavior is structurally generic beyond those families.
