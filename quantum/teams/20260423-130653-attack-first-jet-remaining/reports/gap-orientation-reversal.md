# gap-orientation-reversal

- **Claim**: Let \(\lambda \mapsto |\psi(\lambda)\rangle\) be a \(C^1\) normalized one-parameter pure-state lift, \(h_\lambda=(I-|\psi\rangle\langle\psi|)\partial_\lambda|\psi\rangle\), assume \(\langle h_{\lambda_\pm}\mid h_{\lambda_\pm}\rangle>0\), and fix a two-point transport rule \(U_{-\leftarrow +}\) with fixed endpoint ordering. Form the \(2\times 2\) first-horizontal-jet blocks \(M_\lambda(\lambda_i)\), \(N_\lambda(\lambda_-,\lambda_+)\), and \(\Omega_\lambda\) exactly as in the note. If \(\mu=\phi(\lambda)\) is orientation-reversing, then for the same physical ordered pair of endpoints \((\lambda_-,\lambda_+)\) carried to \((\mu_-,\mu_+)=(\phi(\lambda_-),\phi(\lambda_+))\) without re-sorting, the exact law is
  \[
  \Omega_\mu = J\,\Omega_\lambda\,J,\qquad J=\operatorname{diag}(1,-1).
  \]
  Equivalently: the value channel is unchanged and each derivative channel picks up one minus sign. If instead one reorders endpoints so that \(\mu_-<\mu_+\) in the new coordinate, then the statement becomes \(\Omega_\mu(\mu_-,\mu_+)=J\,\Omega_\lambda(\lambda_+,\lambda_-)\,J\), so the law does depend on endpoint ordering.
- **Status**: proved
- **Evidence**: Proved: the first horizontal jet rescales by the chain rule, \(h_\mu=(d\lambda/d\mu)h_\lambda\), and the same-point block is diagonal because \(\langle\psi\mid h_\lambda\rangle=0\). Proved: for an orientation-reversing reparameterization, \(a_i=(d\lambda/d\mu)|_{\mu_i}<0\), so \(N_\mu=A_-N_\lambda A_+\) with \(A_i=\operatorname{diag}(1,a_i)\), while \(M_\mu(\mu_i)=A_iM_\lambda(\lambda_i)A_i\). Proved: since \(M_\lambda(\lambda_i)\) is diagonal positive, its positive inverse square root transforms as \(M_\mu(\mu_i)^{-1/2}=|A_i|^{-1}M_\lambda(\lambda_i)^{-1/2}\). Because \(A_i=|A_i|J\) when \(a_i<0\), the bilateral whitening gives \(\Omega_\mu=J\Omega_\lambda J\). Conditional: this is a ray-curve statement only after a gauge-covariant transport rule is included as part of the data. Missing: any stronger simplification after re-sorting endpoints, because reversing endpoint order also reverses the ordered transport comparison.

- **Dependencies**: One-dimensional \(C^1\) family; first horizontal jet \(h_\lambda=P\partial_\lambda\psi\); nonvanishing endpoint horizontal norms; diagonal same-point first-jet block; fixed ordered pair of endpoints; fixed two-point transport rule; for the ray-space version, gauge-covariant transport.
- **Strongest objection**: Literal equality fails under orientation reversal; only \(J\)-conjugacy is proved. Also, if one silently reindexes endpoints by the new coordinate order, the ordered pair is changed, so one is no longer comparing the same two-point matrix. Transport conventions can also be orientation sensitive, so no transport-independent endpoint invariant follows from this alone.
- **Needed for promotion**: State the theorem with the endpoint-order convention explicit. Promote the orientation-reversing analogue only in the form “\(\Omega\) is invariant up to derivative-channel sign conjugation \(J\Omega J\), and after re-sorting endpoints one must compare with the reversed ordered pair.”
