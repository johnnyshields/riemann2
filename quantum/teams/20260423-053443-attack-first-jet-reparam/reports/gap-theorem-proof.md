# gap-theorem-proof

- **Claim**: There is a real but narrow theorem. Let \(I\ni \lambda\mapsto |\psi(\lambda)\rangle\) be a \(C^1\) normalized local lift of a one-parameter pure-state family, and set \(|h_\lambda\rangle := (I-|\psi(\lambda)\rangle\langle\psi(\lambda)|)\partial_\lambda|\psi(\lambda)\rangle\). For two endpoints \(\lambda_-,\lambda_+\) with \(\langle h_{\lambda_\pm}\mid h_{\lambda_\pm}\rangle>0\), define
  \[
  M_\lambda(\lambda_i):=
  \begin{pmatrix}
  1&0\\
  0&\langle h_{\lambda_i}\mid h_{\lambda_i}\rangle
  \end{pmatrix},
  \]
  a transported mixed block \(N_\lambda(\lambda_-,\lambda_+)\) in the frame \((|\psi\rangle,|h_\lambda\rangle)\), and \(\Omega_\lambda:=M_\lambda(\lambda_-)^{-1/2}N_\lambda M_\lambda(\lambda_+)^{-1/2}\). Then for any orientation-preserving \(C^1\) reparameterization \(\mu=\phi(\lambda)\), the same construction in the \(\mu\)-coordinate gives \(\Omega_\mu=\Omega_\lambda\). This is a theorem for the first horizontal jet in a fixed local lift; it is not yet a canonical two-point invariant of ray space.
- **Status**: proved
- **Evidence**: Proved: horizontality gives \(\langle\psi(\lambda)\mid h_\lambda\rangle=0\), so each same-point block is diagonal. Proved: under \(\mu=\phi(\lambda)\), with \(a(\mu):=d\lambda/d\mu\), one has \(|h_\mu\rangle=a(\mu)|h_\lambda\rangle\). Hence at each endpoint \(i\), \(M_\mu(\mu_i)=S_iM_\lambda(\lambda_i)S_i\), \(N_\mu=S_-N_\lambda S_+\), with \(S_i=\operatorname{diag}(1,a_i)\), \(a_i>0\). Proved: because \(a_i>0\) and \(M_\lambda(\lambda_i)\) is diagonal, \(M_\mu(\mu_i)^{-1/2}=S_i^{-1}M_\lambda(\lambda_i)^{-1/2}\). Therefore \(\Omega_\mu=\Omega_\lambda\). Proved: the oscillator \(\omega\leftrightarrow r=\tfrac12\log\omega\) calculation is one exact instance of this general cancellation mechanism. Conditional: if one asks for a statement on pure states modulo phase, extra two-point gauge/transport structure is still needed. Missing: any extension of this exact matrix invariance beyond first jet or beyond one parameter.

- **Dependencies**: Chain rule for a one-parameter \(C^1\) reparameterization; projector definition of the first horizontal derivative; fixed smooth local lift \(|\psi(\lambda)\rangle\) on the interval; endpoint nondegeneracy so whitening is defined; orientation-preserving hypothesis \(d\lambda/d\mu>0\).
- **Strongest objection**: The theorem is smaller than the phrase “pure-state invariant” suggests. The raw two-point mixed block still depends on the chosen local phase lift, and an orientation-reversing reparameterization gives sign conjugations on the derivative channel rather than literal equality. So the exact invariant theorem is only: first-horizontal-jet bilateral whitening is reparameterization-invariant for orientation-preserving one-parameter changes in a fixed local lift.
- **Needed for promotion**: State the theorem exactly in that narrow form, with the nonvanishing-endpoint hypothesis. Keep separate, and still unproved, the stronger claim that this is a canonical two-point invariant of ray space; that would require a phase-gauge/transport theorem. For any broader promotion beyond first jet or one parameter, a new proof is needed.
