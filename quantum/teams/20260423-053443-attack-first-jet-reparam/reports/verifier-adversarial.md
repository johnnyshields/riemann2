# verifier-adversarial

- **Claim**: The exact version that survives review is narrow: let \(\lambda \mapsto |\psi(\lambda)\rangle\) be a \(C^1\) normalized pure-state family on a one-dimensional parameter interval, set \(P(\lambda)=I-|\psi\rangle\langle\psi|\), \(|j_\lambda\rangle:=P(\lambda)\partial_\lambda|\psi(\lambda)\rangle\), assume \(\langle j_\lambda|j_\lambda\rangle>0\), and choose one fixed gauge-covariant transport rule \(U_{1\leftarrow 2}\) for two-point comparisons. Form the first-horizontal-jet value/derivative blocks in the frame \((|\psi\rangle,|j_\lambda\rangle)\) and the bilaterally whitened block \(\Omega(\lambda_1,\lambda_2)\). Then under any orientation-preserving reparameterization \(\mu=f(\lambda)\), the whitened first-horizontal-jet block is exactly unchanged. No broader theorem is justified from this scope alone.
- **Status**: proved
- **Evidence**: Proved: the same-point first horizontal block is the exact quantum first-order object, with \(j_\lambda=P\partial_\lambda\psi\) and \(\langle \psi|j_\lambda\rangle=0\), so the same-point value/derivative block is diagonal; this removes the off-diagonal same-point-block risk at first order in one parameter. Proved: under reparameterization, \(P\) is unchanged and \(j_\mu=P\partial_\mu\psi=(d\lambda/d\mu)\,j_\lambda\), so only the derivative channel rescales. Proved: therefore \(M_\mu=D\,M_\lambda\,D\) and \(N_\mu=D_1\,N_\lambda\,D_2\) with positive diagonal endpoint factors, and the bilateral whitening cancels these endpoint factors exactly. Proved: the oscillator calculation is one exact instance of this mechanism. Conditional: this is a theorem only for the one-parameter first-horizontal-jet construction with a fixed gauge-covariant transport rule already chosen. Missing: any proof that the same exact matrix invariance survives orientation reversal without residual sign matrices, survives higher jets where triangular mixing appears, survives multiparameter families, or survives without fixing transport.

- **Dependencies**: One-dimensional \(C^1\) normalized pure-state family; first horizontal jet \(j_\lambda=P\partial_\lambda\psi\); nondegenerate first horizontal norm so whitening is defined; one fixed gauge-covariant two-point transport convention; orientation-preserving reparameterization \(f' > 0\); first-order only.
- **Strongest objection**: The theorem is exact only because all four hazards are neutralized by the narrow setup: gauge dependence is pushed into the externally fixed transport convention, sign/orientation problems are excluded by \(f'>0\), same-point off-diagonal blocks vanish by horizontality, and reparameterization acts on the one-dimensional derivative channel by scalar endpoint rescaling rather than higher-jet triangular mixing. If any of those hypotheses is relaxed, the matrix statement can fail or weaken.
- **Needed for promotion**: Write the theorem as a formal proposition with proof. State explicitly that the surviving theorem is only the one-parameter first-horizontal-jet bilateral-whitening invariance under orientation-preserving reparameterization with fixed transport. Record the orientation-reversing variant precisely, likely as equality only up to endpoint sign matrices. If a stronger claim is wanted, prove one of: a canonical transport rule, a transport-independence theorem, a higher-jet covariantization immune to triangular mixing, or a reason the honest invariant is the full matrix rather than only transported-subspace data.
