# gap-nonconstant-example

- **Claim**: The best first serious small example is the one-parameter family of harmonic-oscillator ground states with variable frequency
  \[
  \psi_\omega(x)=\left(\frac{\omega}{\pi}\right)^{1/4}e^{-\omega x^2/2},\qquad \omega>0,
  \]
  viewed through the first jet pair \((|\psi_\omega\rangle,\partial_\omega|\psi_\omega\rangle)\). It is better than the earlier qubit and coherent-displacement cases because the same-point jet Gram block is exact and nonconstant,
  \[
  M(\omega)=\begin{pmatrix}1&0\\[0.3ex]0&\frac1{8\omega^2}\end{pmatrix},
  \]
  so for two endpoints \(\omega_-\neq \omega_+\) the whitening is genuinely bilateral. The mixed block is still closed form, and the resulting whitened block is an exact stress test for nonconstant same-point blocks and asymmetric left/right whitening.
- **Status**: heuristic
- **Evidence**: Proved: the paper-side target is exactly a two-endpoint construction with two same-point jet Gram blocks and one mixed block, whitened asymmetrically as \(G_{m,-}(s)^{-1/2}N_m(s)G_{m,+}(s)^{-1/2}\). Proved: for the oscillator family, \(\langle\psi_\omega,\partial_\omega\psi_\omega\rangle=0\), so the same-point first-jet block is diagonal with derivative norm \(\|\partial_\omega\psi_\omega\|^2=1/(8\omega^2)\), hence nonconstant. Proved: the endpoint overlap and all first mixed derivative entries are exact derivatives of the closed-form overlap \(I(\omega_-,\omega_+)=\sqrt2\,(\omega_-\omega_+)^{1/4}/\sqrt{\omega_-+\omega_+}\), giving an explicit mixed and whitened block. Conditional: this is a good quantum analogue only if the adopted first-jet construction is the same one already used in the quantum notes, namely state plus horizontal first derivative for a chosen one-parameter family. Missing: a proof that the full matrix, rather than only its singular values or principal-angle data, is canonical under the gauge/reparameterization issues already identified in the earlier cycle.

- **Dependencies**: Accept the one-parameter quantum-side first-jet model \((|\psi(\lambda)\rangle,(I-|\psi\rangle\langle\psi|)\partial_\lambda|\psi\rangle)\). For this real normalized family the projector is harmless because \(\langle\psi_\omega,\partial_\omega\psi_\omega\rangle=0\). Treat \(\omega\) as the chosen control parameter, or equivalently set \(\omega_\pm=m\pm s/2\) to mirror the paper’s symmetric-endpoint notation.
- **Strongest objection**: The nonconstancy is parameter-choice sensitive. If one reparametrizes by the squeeze variable \(r=\frac12\log\omega\), then the derivative norm becomes constant, so this example is an exact stress test for the raw first-jet matrix construction, not a proof of any intrinsic higher-jet invariant. It also has trivial Berry structure because the wavefunctions can be chosen real and positive.
- **Needed for promotion**: Fix the exact quantum-side rule for mixed two-point comparison and its invariance class. Then use this family as the benchmark example in two parallel parameterizations, \(\omega\) and \(r=\frac12\log\omega\), to decide what survives: the full whitened matrix, a frame-dependent transport block, or only singular values/principal angles of the transported jet subspaces.
