# verifier-adversarial

- **Claim**: The strongest safe singular-value statement is narrow: for the one-parameter first-horizontal-jet block \(\Omega_\lambda\), once a gauge-covariant transport rule is fixed as part of the data, the singular values of \(\Omega_\lambda\) are independent of local lift and invariant under every reparameterization of the curve, including orientation reversal. What singular values do not buy, from the present scope alone, is a transport-independent or endpoint-only canonical object. A stronger claim is only conditional: if one restricts to a transport class whose effect on the whitened first-jet frame is unitary on the left/right, in particular the scalar-phase class, then the singular values are transport-class invariant; without that extra restriction, even singular-value canonicity across admissible transports is unproved.
- **Status**: proved
- **Evidence**: Proved: the note establishes \(\Omega_\mu=\Omega_\lambda\) for orientation-preserving reparameterization and \(\Omega_\mu=E\Omega_\lambda E\) for orientation reversal in the one-parameter first-horizontal-jet setting; since \(E\) is unitary, singular values are unchanged in both cases. Proved: with a gauge-covariant transport rule fixed, the block descends from local lifts to ray-curve data, so its singular values do as well. Proved: prior adversarial transport review rules out any unconditional claim that the matrix ambiguity under changing transport is only a global phase; the remaining transport dependence can be entrywise and larger than phase-only, so singular-value invariance across transports does not follow. Conditional: if one narrows the admissible transport class so that changes act by left/right unitary factors on the orthonormalized first-jet frame, then singular values become transport-class invariants; the scalar-phase class is a special case of this.

- **Dependencies**: One-parameter \(C^1\) pure-state ray curve; nonvanishing endpoint first horizontal jets; the first-horizontal-jet \(2\times2\) block; a gauge-covariant transport rule fixed as part of the data; basic fact that singular values are invariant under unitary conjugation and, conditionally, under left/right unitary multiplication.
- **Strongest objection**: Singular values are more canonical than the full matrix only with respect to the already-proved reparameterization and lift ambiguities. From the present scope alone, they do not solve the main unresolved issue, namely transport dependence.
- **Needed for promotion**: Add the split theorem and keep the transport caveat explicit. Do not call the singular values endpoint-canonical or transport-independent outside the reduced transport class.
