# gap-singular-values

- **Claim**: Let \(\lambda \mapsto |\psi(\lambda)\rangle\) be a \(C^1\) one-parameter pure-state ray curve, let \(h_\lambda=(I-|\psi\rangle\langle\psi|)\partial_\lambda\psi\), assume \(\|h_{\lambda_\pm}\|>0\), and form the bilaterally whitened first-horizontal-jet block \(\Omega_\lambda\) from the ordered endpoints \((\lambda_-,\lambda_+)\). Then two exact singular-value statements are safe. First, for a fixed gauge-covariant transport rule included as part of the data, the unordered singular values of \(\Omega_\lambda\) are invariant under local phase gauge and under every reparameterization of the one-dimensional parameter, including orientation reversal. Second, if one explicitly restricts to the reduced scalar-phase transport class \(U_{-\leftarrow +}=e^{i\alpha(\gamma)}I\), then the singular values are also independent of the chosen transport inside that class. So the strongest theorem-sized promoteable statement is: the singular values of the bilaterally whitened first-horizontal-jet block define a first-order invariant of a one-parameter ray curve with ordered endpoints and scalar-phase transport class; without that restriction they are only invariants for a chosen transport rule, not transport-independent endpoint invariants.
- **Status**: proved
- **Evidence**: Proved in the note and reversal report: orientation-preserving reparameterization leaves \(\Omega\) unchanged, while orientation reversal gives \(J\Omega J\); left/right multiplication by the unitary involution \(J\) preserves singular values. Proved in the gauge discussion: for fixed gauge-covariant transport, \(\Omega\) is independent of the local lift, so its singular values are gauge-invariant. Proved in the scalar-phase-class report: if \(U'=e^{i\theta}U\), then the whole whitened block is multiplied by one global phase, which also preserves singular values. Proved in the transport-dependence report: without restricting the transport class, different admissible gauge-covariant transports can change matrix entries by more than a global phase, so no transport-independent singular-value theorem is justified from the present hypotheses alone.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:103-223`; `quantum/findings.md:35-50`; `quantum/findings.md:91-102`; `quantum/20260423-130653-attack-first-jet-remaining/reports/gap-orientation-reversal.md:3-13`; `quantum/20260423-130653-attack-first-jet-remaining/reports/gap-transport-dependence.md:3-17`; `quantum/20260423-131101-attack-transport-class/reports/gap-scalar-phase-class.md:3-9`.
- **Dependencies**: One-parameter \(C^1\) pure-state family; first horizontal jet; nonzero endpoint horizontal norms; fixed ordered endpoints; gauge-covariant transport for the fixed-transport version; the reduced scalar-phase transport class for the transport-independent singular-value version.
- **Strongest objection**: The transport-independent conclusion is not proved for the full class of admissible gauge-covariant transports. Presently that larger class allows more than global-phase freedom, so singular values need not be transport-invariant there.
- **Needed for promotion**: Promote the theorem only in the split form above. State explicitly that the fixed-transport singular-value invariant is proved without extra transport restriction, but that a transport-independent theorem requires the reduced scalar-phase transport class.
