# verifier-adversarial

- **Claim**: The strong replacement claim is rejected: the note cannot safely say that transport can simply be removed while preserving the earlier transport-based horizontal-subspace statement. The exact safe theorem from the present scope is narrower: for a one-parameter normalized pure-state curve in a fixed ambient Hilbert space, the pointwise raw projected-jet osculating subspace
  \[
  O_r(\lambda)=\operatorname{span}\{j_1(\lambda),\dots,j_r(\lambda)\},\qquad j_k=P(\lambda)\partial_\lambda^k\psi(\lambda),
  \]
  is gauge- and reparameterization-invariant as an ambient subspace, and therefore the principal angles between \(O_r(\lambda_-)\) and \(O_r(\lambda_+)\) are transport-free ambient invariants whenever the ranks match. If one instead uses \(\operatorname{span}\{\psi,h\}\) or \(\operatorname{span}\{\psi,j_1,\dots,j_r\}\), that is also an ambient invariant, but it is a different object because it puts the value channel back in.
- **Status**: rejected
- **Evidence**: Proved: the current note records that higher projected jets mix by invertible upper-triangular laws under both phase gauge and one-parameter reparameterization, so the ordered jet list is not canonical but its span is unchanged; this is exactly the safe one-point subspace theorem. Proved: the quartit benchmark shows that once endpoint osculating planes are compared inside a common ambient space, the surviving second-order datum is precisely principal-angle data of those ambient osculating planes. Conditional: in a fixed common ambient Hilbert space, principal angles between endpoint \(O_r\)-subspaces are well defined without transport and are invariant under endpoint phase changes because each endpoint jet list changes only inside its own span. Missing: any proof that this transport-free ambient comparison is equivalent to the earlier transported horizontal-subspace comparison, or that replacing the transport-based language preserves the intended geometric content rather than changing the object.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:198-247,249-299`; `quantum/findings.md:35-54,107-137`; `quantum/20260423-135052-attack-quartit-osculating/reports/gap-quartit-principal-angles.md:3-43`; `quantum/20260423-140111-attack-transport-free-subspaces/notes/coordinator-brief.md:3-16`; `quantum/20260423-134344-attack-higher-jets/reports/gap-higher-jet-theorem.md:3-23`.
- **Dependencies**: Fixed common ambient Hilbert space for all endpoints; one-parameter \(C^r\) normalized local lift; projector \(P=I-|\psi\rangle\langle\psi|\); invertible upper-triangular gauge and chain-rule transformations on \((j_1,\dots,j_r)\); locally constant \(\dim O_r\) when comparing two endpoints; standard linear algebra of principal angles between ambient subspaces.
- **Strongest objection**: The ambient-subspace theorem is not the same as the transport-based theorem. Using ambient \(O_r\) drops the transported tangent-space interpretation; using ambient \(\operatorname{span}\{\psi,h\}\) or \(\operatorname{span}\{\psi,j_1,\dots,j_r\}\) reintroduces the value channel. So from this scope alone, “remove transport” is safe only if the note openly changes the object being studied.
- **Needed for promotion**: State the replacement theorem exactly as an ambient-subspace theorem, not as a transport-free version of the earlier transported matrix/subspace claim. Decide explicitly whether the intended object is \(O_r(\lambda)\) or the larger ambient span including \(\psi(\lambda)\); if the latter, say that it is a different invariant.
