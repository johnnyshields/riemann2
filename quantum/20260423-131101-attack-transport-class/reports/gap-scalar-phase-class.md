# gap-scalar-phase-class

- **Claim**: For the one-parameter first-horizontal-jet theorem, the smallest correct natural transport subclass is the scalar-phase class: along a fixed oriented ray-curve segment, the two-point transport acts on the ambient Hilbert space only by a phase factor \(U_{-\leftarrow +}=e^{i\alpha(\gamma)}I\), with the usual gauge-covariance law under endpoint lift changes. In this subclass the bilaterally whitened first-horizontal-jet block satisfies \(\Omega(U')=e^{i\theta}\Omega(U)\) for any two admissible scalar-phase transports \(U,U'\). So the remaining ambiguity is exactly one global phase on the whole \(2\times2\) block, not entrywise freedom.
- **Status**: proved
- **Evidence**: Proved: the first-horizontal-jet block has the form \(\Omega=M_-^{-1/2} N M_+^{-1/2}\), where every entry of \(N\) is a transported overlap involving \(|\psi_+\rangle\) and \(|h_+\rangle\). Proved: under phase gauge, \(|h_\lambda\rangle\mapsto e^{i\chi(\lambda)}|h_\lambda\rangle\), so every mixed entry picks up the same endpoint phase factor when \(U\) is scalar-phase. Proved: if \(U'=e^{i\theta}U\), then every entry of the mixed block, hence \(\Omega\) itself after bilateral whitening, is multiplied by the same \(e^{i\theta}\). Conditional: Berry gives a full oriented-curve phase transport; Pancharatnam gives an endpoint phase matching rule when endpoint overlap is nonzero. On such segments they lie in the same scalar-phase class and therefore produce whitened blocks differing only by one global phase. Missing: a deposited proof that every intended “natural” transport must reduce to this scalar-phase subclass, and a zero-overlap extension of the Pancharatnam comparison.
- **Exact refs**: `quantum/paper/jet_gram_quantum_note.md:115-205`; `quantum/findings.md:35-49`; `quantum/findings.md:91-98`; `quantum/20260423-130653-attack-first-jet-remaining/reports/gap-transport-dependence.md:3-17`; `quantum/20260423-053832-attack-first-jet-gauge/reports/gap-gauge-proof.md:3-9`; `quantum/20260423-131101-attack-transport-class/notes/coordinator-brief.md:6-14`.
- **Dependencies**: One-parameter \(C^1\) pure-state ray curve in a fixed Hilbert space; nonvanishing endpoint first horizontal jets; fixed ordered endpoints; scalar-phase transport law on the oriented curve segment; gauge-covariance of the transport phase; nonzero endpoint overlap if Pancharatnam is invoked.
- **Strongest objection**: The exact global-phase conclusion is proved only after restricting to the scalar-phase subclass. The current project files do not yet prove that all intended admissible transports, or even all Berry/Pancharatnam-style conventions in singular cases, are forced into that subclass.
- **Needed for promotion**: Prove that the intended canonical transport on a one-parameter ray curve is uniquely scalar-phase, or explicitly promote the theorem in the reduced form: for scalar-phase transports, the bilaterally whitened first-horizontal-jet block is defined up to one global phase.
