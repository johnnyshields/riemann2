# verifier-adversarial

- **Claim**: Restricting to scalar-phase transports is not mathematically natural from the present scope alone. The strongest safe transport-class statement for the note is only conditional: if one explicitly narrows to transports that preserve the whitened first-horizontal-jet frame up to phase, then the residual ambiguity drops from entrywise matrix dependence to diagonal phase freedom; and only under the still stronger extra axiom that the same phase acts on both the ray line and the normalized horizontal line does the whole \(2\times2\) block become well defined only up to a single global phase. Berry and Pancharatnam can serve as candidate phase choices on the state line, but from the present scope alone neither gives a canonical transport law for the derivative channel, so no unconditional transport-independent matrix statement should be added.
- **Status**: proved
- **Evidence**: Proved: the current deposited theorem gives only a ray-curve-with-chosen-transport invariant, not a transport-independent endpoint matrix invariant, and the remaining ambiguity under a bare gauge-covariant transport can be larger than a global phase and can change derivative-channel entries entrywise. Proved: after whitening, the first-jet block is the overlap matrix of the orthonormalized endpoint frame \((\psi_\pm,e_\pm)\), so if one imposes a narrower class of transports carrying the ray line to the ray line and the normalized horizontal line to the horizontal line, the matrix must become diagonal-phase on that frame; if one further imposes a common scalar on both lines, the ambiguity reduces to one global phase. Conditional: Berry transport is a natural curve-based phase convention for the state line, and Pancharatnam is a natural endpoint-phase convention when the endpoint overlap is nonzero, but from the present scope alone there is no proof that either canonically determines the derivative-line transport or that both conventions land in the same admissible class for the full first-jet frame. Missing: a deposited theorem that the scalar-phase class is the correct natural class, a proof that Berry or Pancharatnam uniquely selects it in this first-jet setting, and a proof that the derivative-channel phase is forced rather than chosen.
