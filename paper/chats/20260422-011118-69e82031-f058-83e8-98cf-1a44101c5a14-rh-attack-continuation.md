F(u):=\frac{1}{\widehat Q'(u)},
\]
one has
\[
\Delta'(q)=F(\phi_2(q))-F(\phi_0(q)),
\qquad
\frac{dF}{dq}=-\Xi.
\]
So a same-value / same-slope witness is exactly a self-intersection of the graph
\[
\Gamma(u)=\bigl(\widehat Q(u),\,1/\widehat Q'(u)\bigr)
\]
across the two outer branches. That is a very clean geometric recoding of the transport issue. 

The transport-jet hierarchy is also still sound and useful:
\[
\Xi_0:=\Xi,\qquad
\Xi_{n+1}:=\frac{\Xi_n'}{\widehat Q'},
\]
with identities
\[
\Delta^{(n+2)}(q)=\Xi_n(\phi_0(q))-\Xi_n(\phi_2(q)).
\]
So a higher-multiplicity same-value/same-slope coincidence is exactly failure of separation by a finite transport-jet state. This meshes well with the generator audit: if there is no second primitive generator beyond the septic one, then the first remaining obstruction should look like jet-coincidence failure, not a brand new explicit scalar. 

The Sturm normalization is also still a goodie:
\[
\mathcal K(q):=\frac{\Xi(\phi_2(q))-\Xi(\phi_0(q))}{\phi_2(q)-\phi_0(q)},
\qquad
\Delta''+\mathcal K\,\Delta=0.
\]
So the strict three-run transport lane is, in principle, a one-scalar Sturm-coefficient problem. 

## 5. Conditional but sharp transport endgames that are still worth keeping

I do **not** think these are theorem-level yet, but they remain good endgames.

The strongest conditional transport reduction is first-order inverse-velocity closure:
\[
\Xi(u)=H\!\left(\widehat Q(u),\frac{1}{\widehat Q'(u)}\right).
\]
If that holds on the two outer branches, then \(F=1/\widehat Q'\) satisfies the same first-order ODE in value-space on both branches, so one interior same-value / same-slope witness forces coincidence of the two \(F\)-graphs, hence constant-gap transport:
\[
\phi_2(q)-\phi_0(q)\equiv C.
\]
So under first-order closure, one witness rigidifies the entire overlap transport to exact translated replication. This is still only conditional, but it is a strong and clean fork.  

Similarly, if \(\Xi\) descended purely to value-space across the two branches, then
\[
\Delta''\equiv 0,
\]
so \(\Delta\) would be affine. That is probably too strong to hope for, but it remains a valid closure target. 

A weaker transport endgame is one-sided sign control of
\[
\Upsilon(q)=\Xi(\phi_0(q))-\Xi(\phi_2(q)).
\]
If \(\Upsilon\) has fixed sign, then \(\Delta'\) is monotone, and the transport problem reduces to endpoint sign compatibility. That is a much weaker ask than full descent and still a plausible route. 

So the transport “goodies” I would keep are:

the inverse-velocity graph;
the jet tower \(\Xi_n\);
the Sturm coefficient \(\mathcal K\);
first-order closure as a rigid-translation fork;
and fixed-sign \(\Upsilon\) as a monotonicity/one-crossing endgame.

in the finite branch algebra \(A\). There is also a useful special-fiber contact-order test before the full norm/resultant stage. This is a very elegant reduction, but it is still downstream of the unproved local edge-law package, so I would keep it out of the theorem-level core for now.
