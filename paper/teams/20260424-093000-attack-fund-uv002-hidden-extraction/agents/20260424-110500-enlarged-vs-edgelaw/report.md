Claim

An enlarged package theorem does not subsume Bottlenecks A-B. It can at best replace the current reduced-`\widehat\Psi` formulation of Bottlenecks C-D by a stronger package-state theorem, but it still depends on independent local support input from the good-patch edge law and the exact fixed-shear corner law.

Status

heuristic

Evidence

- The queue documents already separate the fronts: the good-patch edge law and exact fixed-shear package theorem are listed as distinct support theorems beneath `UV-002`, not as consequences of the package/extraction bridge. This separation is repeated in both the team `collation.md` and the frozen convergence note.
- In the draft, the good-patch frontier is a local theorem about the actual corrected cubic/quintic defects in the collision-cancellation blow-up chart: analytic swap-evenness plus diagonal vanishing give the quadratic/projective factorization and the edge traces leading to the detector `M/c` on `J\Subset\{c\neq0,M\neq0\}`. That is Bottleneck A, and it is about local defect asymptotics, not about choosing a larger package state.
- The exact fixed-shear corner is treated separately: after quotient descent, finite quotient-visible transport is rank one in `Q`, but closure still requires either state-locality of the actual corrected defects or the stronger merger law forcing diagonal collapse on the involutive branch. The draft explicitly says any surviving obstruction there is relational/provenance-sensitive two-atom data beyond the descended transport state. A larger package may be the right state for this, but the closure theorem itself is still a separate local package law, i.e. Bottleneck B rather than something eliminated by D.
- The draft also says the honest order-7 target is a provenance-sensitive package statement such as same reduced image germ / collision-functoriality, and the current notes sharpen this further: reduced `\widehat\Psi` is too small, so one may need a larger package state (for example adding the hidden `\lambda`-shear class). That directly addresses Bottlenecks C-D, not A-B.
- Therefore the enlarged-package move changes what the package-side coincidence and hidden extraction theorem should state, but it does not by itself prove the local diagonal vanishing / merger statements needed on the good patch or on the fixed-shear corner.

Dependencies

- A precise enlarged package state theorem replacing bare reduced-`\widehat\Psi` on Bottlenecks C-D.
- A separate theorem showing the actual corrected cubic/quintic defects factor through that state with the required diagonal-collapse / merger law on the fixed-shear involutive branch.
- A separate good-patch edge-law theorem giving the quadratic/projective factorization and edge traces on `J\Subset\{c\neq0,M\neq0\}`.

Strongest objection

If one proved an extremely strong enlarged package theorem that already encoded the actual corrected defect package itself, together with diagonal merger on coincidence and the good-patch edge traces, then A-B would become corollaries. But that is no longer just an enlarged version of Bottlenecks C-D; it is effectively the missing local support theorems bundled into the package statement.

Needed for promotion

- State the enlarged package theorem in a form that explicitly distinguishes what it replaces (reduced-`\widehat\Psi` coincidence / hidden extraction) from what it still assumes (local defect laws).
- Prove whether the actual corrected cubic/quintic defects factor through the enlarged state on the fixed-shear corner and vanish under diagonal merger.
- Prove or retain the good-patch edge-law theorem separately unless its edge traces are built into the enlarged theorem itself.

Honest verdict: the enlarged package theorem is a sharpened Bottleneck C-D object, not a genuine replacement for Bottlenecks A-B. The local edge-law and fixed-shear support fronts still carry independent content unless one strengthens the package theorem so far that it already contains those local laws.
