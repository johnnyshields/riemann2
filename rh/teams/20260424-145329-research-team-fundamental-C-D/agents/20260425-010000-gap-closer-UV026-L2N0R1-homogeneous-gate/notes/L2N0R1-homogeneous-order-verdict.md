# L2N0R1 / L1N0R2 Homogeneous Order Verdict

Separate proved / conditional / missing.

## Order Verdict

Under the homogeneous scalar grade
\[
\operatorname{Gr}_a r
=\frac{r^{(a+2)}(m)}{(a+2)!}(t-m)^{a+2},
\]
the same-point first-Frechet grade-\(a\) input starts at ordinary \(z\)-order
\(a\), and a second-Frechet term with grade multiset \(\{a,b\}\) starts at
order \(a+b\).  The baseline mixed block \(N_0\) is holomorphic and has an
order-zero coefficient.

Therefore the non-Y families are not removed by order count:

1. \(\{1,1\}\) second Frechet with grade \(5\) first Frechet:
\[
2+0+5=7;
\]
2. \(\{1,5\}\) second Frechet with grade \(1\) first Frechet:
\[
6+0+1=7.
\]

The mirror family \(L_1N_0R_2\) has the same order counts.

## Sharpened Missing Theorem

Because each live term lands exactly at \(z^7\), only leading coefficients are
needed.  For \(L_2N_0R_1\), compute
\[
[z^2]\mathcal L_-^{\{1,1\}},\quad
[z^6]\mathcal L_-^{\{1,5\}},\quad
[z^0]N_0,\quad
[z^5]\Lambda_+^{[5]},\quad
[z^1]\Lambda_+^{[1]}.
\]
For \(L_1N_0R_2\), compute the mirrored list with \(\Lambda_-\) and
\(\mathcal R_+\).

Higher coefficients of \(N_0\), of the Frechet outputs, or of the baseline
inverse square roots cannot enter these homogeneous-grade \(B_7\) terms.

## Remaining Caveat

This is still not a fixed-sector gauge proof.  The four leading matrix sums
must be computed before any gauge or determinant test is meaningful.
