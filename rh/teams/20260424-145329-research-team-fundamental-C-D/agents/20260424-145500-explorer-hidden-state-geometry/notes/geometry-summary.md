# Hidden-state geometry notes

1. In the fixed-sector basis \\(I,S,D,K\\), the functional \\(\Phi_K(X)=x_{12}+x_{21}\\) is exactly the coefficient extractor for the \\(S\\)-direction. Hence
   \
   \[
   \ker\Phi_K=\mathbf C I\oplus \mathbf C D\oplus \mathbf C K.
   \]

2. The canonical order-7 datum already present in the paper is not a raw septic vector \\((u_7,v_7)\\), but the quotient class \\([A_7^{\mathfrak f}]\in \mathfrak f/\mathbf C A_5^{\mathfrak f}\\), equivalently the determinant scalar
   \
   \[
   \Delta_7=u_7v_5-u_5v_7.
   \]
   The gauge law is translation by the quintic line:
   \[
   A_7^{\mathfrak f}\mapsto A_7^{\mathfrak f}+c_2A_5^{\mathfrak f}.
   \]

3. Therefore the residual hidden degree of freedom is geometrically an affine coordinate along the line \\(\mathbf C A_5^{\mathfrak f}\\) inside the fixed-sector plane \\(\mathfrak f=\operatorname{span}\{I,S\}\\). After fixing \\(A_5^{\mathfrak f}\\), the canonical quotient remembers the transverse class, but forgets motion parallel to \\(A_5^{\mathfrak f}\\).

4. Since \\(\Phi_K\\) reads the \\(S\\)-component, a gauge translation by \\(c_2A_5^{\mathfrak f}=c_2(u_5I+v_5S)\\) changes the visible scalar by \\(c_2v_5\\). So the hidden line is not automatically invisible to \\(\Phi_K\\); invisibility requires an additional theorem forcing equality of the affine-line coordinate, or forcing the package difference into \\(\ker\Phi_K\\).

5. Structural consequence: Bottleneck D is not merely about quotienting out the order-7 gauge. It must identify a canonical lift, or show that the first surviving odd coefficient depends only on the quotient-level package despite the potentially visible line direction.
