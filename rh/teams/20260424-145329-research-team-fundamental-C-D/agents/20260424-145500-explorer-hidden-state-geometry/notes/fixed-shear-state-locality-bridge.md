# Fixed-shear state-locality bridge to the hidden lift picture

1. The fixed-shear section already contains a descended transport-state formalism on the involutive branch
   \[
   (h_1,h_2)=(h(t),h(-t)),\qquad u=t^2,
   \]
   with parity-normalized jets
   \[
   \Theta_k(t),\qquad \mathcal T_N(t)=(\Theta_0(t),\dots,\Theta_N(t)).
   \]
   Every \(\Theta_k\) is even analytic, so every finite quotient-visible transport state descends to the quotient variable \(u=t^2\).

2. On the quartic--sextic rung, every finite quotient-visible transport state is a function of the single scalar
   \[
   Q(t)=\widehat Q(t^2),
   \]
   and there is no third local reset beyond the involution \(t\leftrightarrow -t\).
   So the quotient-visible transport side already has trivial local monodromy.

3. The missing theorem is then not another transport coordinate. It is state-locality of the actual corrected two-pair defect/package with respect to this descended state. Concretely, if
   \[
   E_{12}^{(3)}(t)=\mathfrak e_3(\mathcal T_N(t)),\qquad E_{12}^{(5)}(t)=\mathfrak e_5(\mathcal T_N(t)),
   \]
   and these vanish at coincidence, then evenness forces
   \[
   E_{12}^{(3)}(t),E_{12}^{(5)}(t)=O(t^2),
   \]
   which excludes the residual branch.

4. Stronger still, if the full corrected two-pair package is state-local in the descended transport state and satisfies the natural merger law
   \[
   \mathfrak P_2(a_1,a_2;Y,Y)=(a_1+a_2)\mathfrak F(Y),
   \]
   then the interaction remainder vanishes identically on the involutive branch.

5. This matches the affine-lift interpretation. The affine hidden coordinate can survive only if the actual two-atom package fails to factor through the descended quotient-visible transport state. In affine-bundle language: the bundle already has trivial quotient-visible monodromy; the unresolved issue is whether the corrected package depends only on the descended base state or also on extra relational/provenance-sensitive lift data.

6. Therefore the cleanest reformulation of D-min is not merely “find a canonical section.” It is:
   - prove the actual corrected two-atom package is state-local in the descended transport state; or
   - isolate the first genuinely relational component that obstructs such state-locality.

7. The strongest paper-backed negative statement here is scoped: from the quotient-visible descended state alone, there is no finite-order hidden reset and no second local transport variable. Any still-live hidden state must therefore be genuinely relational/provenance-sensitive or non-finite-order.
