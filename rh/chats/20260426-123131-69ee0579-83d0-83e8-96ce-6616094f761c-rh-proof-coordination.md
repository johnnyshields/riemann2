```latex
\begin{theorem}[Conditional source--anisotropy bridge]

\[
u=q'',\qquad

Assume the following good-patch bounds hold on a fixed enlargement of \(I_\rho\):
\[
|q_0|+|u|+|q^{(4)}|+|G|\le Q^C,

\[
|H|\ge Q^{-C_H}|u|^3

at all points where the rank-two rows are evaluated. Suppose further that the vector source--anisotropy estimate
\[
\int_{I_\rho}|u(m)|^2|G(m)|^2\,dm
\ge Q^{-A_v}|I_\rho|

holds. Then
\[
\sup_{m\in I_\rho}
|H(m)|\max(|\Gamma_G(m)|,|\Gamma_{G,2}(m)|)
\ge Q^{-A}
\]
for some \(A=A(A_v,C,C_H)\).
\end{theorem}
```

Proof sketch:

From

\[
\int_I |u|^2|G|^2\ge Q^{-A_v}|I|
\]

we get

\[
\sup_I |u||G|\ge Q^{-A_v/2}.
\]

Using

\[
|G|\le Q^C,
\]

at such a point,

\[
|u|\ge Q^{-A_v/2-C}.
\]

Then

\[
|H||G|
\ge
Q^{-C_H}|u|^3|G|
=
Q^{-C_H}|u|^2(|u||G|)
\ge
Q^{-C_H-2(A_v/2+C)-A_v/2}.
\]

Thus the original rank-two row lower bound follows polynomially.

So the real missing step is not this bridge. The missing step is the **vector source-anisotropy estimate**.

## 1. Natural global energy identity

## 3. Beurling-Selberg / Weil-positive test

The second possible escape is to use a Weil-positive test.

A hypothetical theorem would look like this:

\[
\mathcal W_f(\zeta)
=
\sum_\rho H_f(\rho)
-
\mathcal P_f
-
\mathcal G_f
\]

where \(f\) is a carefully localized test function, \(\widehat f\) has controlled sign/support, \(\mathcal P_f\) is the prime side, and \(\mathcal G_f\) is the gamma/background side.

One would want:

\[
\text{off-line packet}+q''_{\can}\text{-flatness}
\Longrightarrow
\mathcal W_f<0
\]

while the prime/gamma side forces

\[
\mathcal W_f\ge0.
\]

This is the right global shape, but it is essentially a Weil-criterion route. Unless the test function is constructed from some new packet-visible invariant, it is not easier than RH. It risks becoming:

\[
\text{choose a test detecting an off-line zero}
\Longrightarrow
\text{contradict Weil positivity}.
\]

But Weil positivity for all admissible tests is itself RH-equivalent. So this is a valid **global pivot**, not a local rescue theorem.

The only potentially useful refinement would be a **restricted Weil positivity theorem** for the specific packet-adapted family of tests produced by source-flatness. That would be new and meaningful. But I do not see a currently available argument proving that restricted positivity from standard tools.

That is Agent 4’s gate, not a bypass.

```python
def classify_interval(row, thresholds=None):
    """
    Classify one Gate 1 interval record.

    Required row keys:
      Q, E_I, S_I, B_eff, Jmax, J2, R
      D_tail, D_quad, D_reg, D_width
      D_AFE optional / None
      J0 optional
      E_control optional
      Jmax_control optional
      S_control optional
    """

    Q = row["Q"]
    B_eff = row["B_eff"]
    Jmax = row["Jmax"]

    D_tail = row.get("D_tail", None)
    D_quad = row.get("D_quad", None)
    D_reg = row.get("D_reg", None)
    D_width = row.get("D_width", None)
    D_AFE = row.get("D_AFE", None)

    reasons = []
    flags = []

    # Hard rejection checks
    if D_tail is not None and D_tail >= 1.0:
        return "reject_tail_artifact", ["tail drift >= 1"]

    if D_quad is not None and D_quad >= 1.0:
        return "reject_quadrature_artifact", ["quadrature drift >= 1"]

    if D_reg is not None and D_reg >= 1.0:
        return "reject_regularization_artifact", ["regularization drift >= 1"]

    if D_AFE is not None and D_AFE >= 1.0:
        return "reject_AFE_disagreement", ["AFE disagreement >= 1"]

    if D_width is not None and D_width >= 1.0:
        flags.append("width_fragile")

    # Isolated point smallness
    J0 = row.get("J0", None)
    if J0 is not None:
        if B_eff <= 1.5 and J0 <= Q**(-5):
            return "reject_point_small_only", [
                "J0 is small but interval energy is ordinary"
            ]

    # Stability helpers
    def ok(value, limit):
        return value is None or value < limit

    pilot_stable = (
        ok(D_tail, 0.25) and
        ok(D_quad, 0.25) and
        ok(D_reg, 0.25) and
        ok(D_width, 0.50) and
        ok(D_AFE, 0.50)
    )

    serious_stable = (
        ok(D_tail, 0.10) and
        ok(D_quad, 0.10) and
        ok(D_reg, 0.10) and
        ok(D_width, 0.25) and
        ok(D_AFE, 0.25)
    )

    # Classify
    if B_eff <= 1.5:

    if B_eff <= 3.0 or Jmax > Q**(-3):

                "strong flatness, stable, but no AFE validation"

            "strong interval and jet flatness with stability"
        ]
```

# 4. Proxy artifact risk

Any run using artificial critical-line regularization

\[
c=0\rightsquigarrow \varepsilon=\kappa/Q
\]

gets a `proxy_artifact_risk`.

Define:

\[
R_{\rm reg}:=\min(1,D_{\rm reg}).
\]

Define:

\[
R_{\kappa}:=
\begin{cases}
1, & \text{if only one }\kappa\text{ was tested},\\
0, & \text{if at least four }\kappa\text{ values were tested and }D_{\rm reg}<0.10,\\
0.5, & \text{otherwise}.
\end{cases}
\]

Define:

\[
R_{\rm canon}:=
\begin{cases}
1, & \text{proxy-only zero kernel},\\
0.5, & +B_2\text{ but no canonical critical-line treatment},\\
0.25, & +B_2\text{ and critical regularization but no tail correction},\\
0, & \text{canonical zero-side object}.
\end{cases}
\]

Then:

\[
\texttt{proxy\_artifact\_risk}
=
\min(1,\ 0.4R_{\rm reg}+0.3R_{\kappa}+0.3R_{\rm canon}).
\]

Classification rule:

- If `proxy_artifact_risk >= 0.7`, do not allow `serious_warning_AFE_validated`.
- If `proxy_artifact_risk >= 0.5`, at most classify as `candidate_flat` unless AFE agreement is strong.
- If `D_reg >= 1`, reject as `reject_regularization`.

# 5. Classifier pseudocode

```python
def classify_interval(row):
    Q = row["Q"]
    B_eff = row["B_eff"]
    Jmax = row["Jmax"]
    J0 = row.get("J0")

    D_tail = row.get("D_tail")
    D_quad = row.get("D_quad")
    D_reg = row.get("D_reg")
    D_width = row.get("D_width")
    D_AFE = row.get("D_AFE")

    reasons = []

    def present(x):
        return x is not None and not is_nan(x)

    # Hard rejections
    if present(D_tail) and D_tail >= 1.0:
        return "reject_tail", ["tail drift >= 1"]

    if present(D_quad) and D_quad >= 1.0:
        return "reject_quadrature", ["quadrature drift >= 1"]

    if present(D_reg) and D_reg >= 1.0:
        return "reject_regularization", ["regularization drift >= 1"]

    if present(D_AFE) and D_AFE >= 1.0:
        return "reject_AFE_disagreement", ["AFE disagreement >= 1"]

    if present(D_width) and D_width >= 1.0:
        return "reject_width_fragile", ["width drift >= 1"]

    # Isolated point smallness
    if present(J0):
        if abs(J0) <= Q**(-5) and B_eff <= 1.5:
            return "reject_point_small_only", [
                "J0 small but interval energy ordinary"
            ]

    # Healthy
    if B_eff <= 1.5:
        return "healthy", ["ordinary interval energy"]

    # Stability gates
    first_pass_stable = (
        (not present(D_tail) or D_tail < 0.25) and
        (not present(D_quad) or D_quad < 0.25) and
        (not present(D_reg) or D_reg < 0.25) and
        (not present(D_width) or D_width < 0.50)
    )

    serious_stable = (
        (not present(D_tail) or D_tail < 0.10) and
        (not present(D_quad) or D_quad < 0.10) and
        (not present(D_reg) or D_reg < 0.10) and
        (not present(D_width) or D_width < 0.25)
    )

    afe_available = present(D_AFE)
    afe_good_candidate = (not afe_available) or D_AFE < 0.50
    afe_good_serious = (not afe_available) or D_AFE < 0.25

    proxy_risk = row.get("proxy_artifact_risk", 0.0)

    # Serious warnings
    if B_eff > 5.0 and Jmax <= Q**(-5) and serious_stable and afe_good_serious:
        if afe_available and proxy_risk < 0.7:
            return "serious_warning_AFE_validated", [
                "strong interval energy suppression",
                "strong jet suppression",
                "stability passed",
                "AFE agreement passed",
            ]
        else:
            return "serious_warning_zero_side_only", [
                "strong interval energy suppression",
                "strong jet suppression",
                "stability passed",
                "AFE missing or proxy risk too high",
            ]

    # Candidate flat
    if B_eff > 3.0 and Jmax <= Q**(-3) and first_pass_stable and afe_good_candidate:
        return "candidate_flat", [
            "candidate interval energy suppression",
            "candidate jet suppression",
            "first-pass stability passed",
        ]

    # Watch
    return "watch", [
        "some suppression or instability, but candidate criteria not met"
    ]
```

# 6. Family detection

A route-threatening family should be hard to trigger.

A family requires:

1. At least five intervals with classification:
   \[
   \texttt{serious\_warning\_zero\_side\_only}
   \]
   or
   \[
   \texttt{serious\_warning\_AFE\_validated}.
   \]

2. At least three distinct height bands.

3. Rerun confirmation for all five intervals.

4. At least two interval center families, unless all candidates are actual off-line/synthetic centers.

5. No common rejection mechanism:
   - no shared regularization artifact;
   - no shared tail cutoff artifact;
   - no shared quadrature artifact.

6. If AFE exists, at least three intervals must be AFE-validated.

Pseudocode:

```python
def classify_family(rows):
    serious = [
        r for r in rows
        if r["classification"] in {
            "serious_warning_zero_side_only",
            "serious_warning_AFE_validated",
        }
    ]

    if len(serious) == 0:
        return "no_family"

    height_bands = set(r["height_band"] for r in serious)
    center_types = set(r["center_type"] for r in serious)
    rerun_pass = [r for r in serious if r.get("rerun_pass", False)]
    afe_valid = [
        r for r in serious
        if r["classification"] == "serious_warning_AFE_validated"
    ]

    if len(serious) >= 5 and len(height_bands) >= 3 and len(rerun_pass) >= 5:
        if len(center_types) >= 2 and len(afe_valid) >= 3:
            return "route_threatening_family"
        return "family_watch"

    if len(serious) >= 2:
        return "family_watch"

    return "isolated_warning"
```
