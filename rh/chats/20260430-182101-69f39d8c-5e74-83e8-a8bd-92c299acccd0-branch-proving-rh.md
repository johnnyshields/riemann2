where \(c^+_{r,k,\ell}\) is a linear combination of derivatives \(B_H^{(k)}(s)\), \(0\le k\le3\). Similarly, the dual half contributes terms

\[
c^-_{r,k,\ell}(s)
(\log n)^r
n^{s-1}
\partial_s^\ell V_-(nX;s),
\qquad
0\le r+k+\ell\le3,
\]

with \(c^-_{r,k,\ell}\) involving derivatives of \(B_H(s)\chi(s)=B_H(1-s)\).

Therefore

\[
u_H(t)
=
F_X^{H,3}(t)+R_X^{H,3}(t),
\]

where

\[
\begin{aligned}

&=
\sum_{0\le r+k+\ell\le3}
\sum_{n\ge1}
a^+_{r,k,\ell}(t)
(\log n)^r
n^{-1/2-it}
V^+_{r,k,\ell}(n/X;t)
\\
&\quad+
\sum_{0\le r+k+\ell\le3}
\sum_{n\ge1}
a^-_{r,k,\ell}(t)
(\log n)^r
n^{-1/2+it}
V^-_{r,k,\ell}(nX;t)
+
P_{\res,X}^{H,3}(t).
\end{aligned}
\]

Here:

\[
a^\pm_{r,k,\ell}(t)
\]

are smooth gamma/theta derivative amplitudes built from \(B_H\), \(B_H'\), \(B_H''\), \(B_H'''\), and the differentiated functional-equation factor.

The residue term \(P_{\res,X}^{H,3}\) contains all differentiated contour residues. It must be retained in the finite main term unless separately shown perturbative.

For every fixed derivative order \(J\), standard smoothed AFE contour estimates and Stirling bounds give

\[
\|\partial_t^J R_X^{H,3}\|_{L^2(W_{\ref})}
+
\|\partial_t^J R_X^{H,3}\|_{L^2(W_{\branch}^{\can})}
\le Q^{-A_{\tail}}
\]

after choosing the cutoff depth sufficiently large relative to the polynomial losses.

This discharges the differential Hardy AFE requirement.

## 2. Dyadic mean-square lower bound

The standard weighted Hardy-derivative mean-square theorem gives, for fixed \(k\),

\[
\int_{\mathbb R}
|Z_H^{(k)}(t)|^2
W(t/T)\,dt
=
T\,P_{2k+1}(\log T)
+
O_{k,W}\!\left(T(\log T)^{2k}\right),
\]

where \(P_{2k+1}\) is a real polynomial of degree \(2k+1\) with positive leading coefficient, provided \(W\ge0\), smooth, compactly supported in a dyadic interval, and not identically zero.

For \(k=3\),

\[
\int
|\partial_t^3Z_H(t)|^2
W(t/T)\,dt
=
c_{3,W}T(\log T)^7
+
O_W(T(\log T)^6),
\qquad
c_{3,W}>0.
\]

Thus, for sufficiently large \(T\),

\[
\int
|u_H(t)|^2
W(t/T)\,dt
\ge
cT(\log T)^7.
\]

For a canonical dyadic reference weight \(W_{\ref}\) comparable to such a \(W(t/T)\),

\[
\int_{\Omega_T}
|u_H(t)|^2W_{\ref}(t)\,dt
\ge
Q^{-A_{\ms}}G_{\ref}.
\]

The exponent \(A_{\ms}\) absorbs:

\[
W_{\ref}\text{-comparison loss},
\qquad
Q\le T^{C_Q},
\qquad
T\text{-threshold loss},
\qquad
(\log T)^7\ge Q^{-o(1)}.
\]

Important scope point: this is not a lower bound for arbitrary microscopic or adversarial windows. It is a dyadic canonical reference-window theorem. If \(W_{\ref}\) is allowed to concentrate near zeros of \(Z_H^{(3)}\), the statement is false without an added nonconcentration hypothesis.

\to

\to

\to

\ge

This uses only:

\[
\text{core covering},
\qquad
w\ge c_w,
\qquad
\text{scale-slow }W_{\ref}.
\]

It does not use the desired \(L^2\) lower bound.

If the branch window were instead defined only over mass-good packets, pointwise domination could fail. The safe formulation is: full-cover \(W_{\branch}^{\can}\) gives pointwise domination; mass-good selection then preserves polynomial source energy.

Yes, inside the ordinary Hardy-normalized canonical branch.

\text{canonical dyadic }W_{\ref}

\text{Whitney core cover}

\text{scale-slow reference weight}

That remains Agent 1’s left-edge entry problem. Agent 2’s source theorem starts only after the ordinary Hardy branch and canonical reference window have already been supplied.
